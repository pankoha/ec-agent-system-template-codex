# Behavioral Pricing Strategy（行動経済学的価格設計）詳細リファレンス

## 定義
心理学・行動経済学の知見を価格設計に応用し、顧客の購買意思決定を最適化する戦略。
「合理的経済人」ではなく「現実の人間」の意思決定プロセスに基づく価格設計。

## 価格設計テクニック

### 1. アンカリング価格
```
【基本原則】
高い価格を先に見せることで、次の価格が安く感じる

【実装例】
├── 通常価格¥10,000 → 特別価格¥6,980
├── 松竹梅の「松」を最初に表示
├── 「最大50%OFF」をバナーで先に表示
└── 「定価」「メーカー希望小売価格」の明示

【効果測定】
├── 通常価格表示あり: CVR +15-25%
└── 割引率表示: CVR +10-20%
```

### 2. 端数価格（チャームプライシング）
```
【基本原則】
¥3,000より¥2,980の方が安く感じる（左端効果）

【実装パターン】
├── 9エンド: ¥2,980、¥9,800（割安感）
├── 0エンド: ¥3,000、¥10,000（高品質感）
├── 8エンド: ¥2,980、¥9,800（中国市場で縁起良い）
└── 7エンド: ¥2,970（さらに安く見える）

【戦略的残高設計】
├── ¥3,000チャージ → ¥2,980商品購入 → ¥20残高
├── ¥5,000チャージ → ¥4,780商品購入 → ¥220残高
└── 目的: 追加購入への誘導、Breakage収益
```

### 3. デコイ効果（おとり効果）
```
【基本原則】
選ばれにくい選択肢を加えることで、狙った選択肢に誘導

【実装例】
├── Small: ¥300（200ml）
├── Medium: ¥500（400ml）← 狙い
└── Large: ¥480（350ml）← デコイ

【EC応用】
├── プランA: 基本機能 ¥980
├── プランB: 全機能 ¥2,980 ← 狙い
└── プランC: 基本+1機能 ¥2,480 ← デコイ
```

### 4. バンドル価格
```
【基本原則】
単品合計より安いセット価格で客単価UP

【設計パターン】
├── Pure Bundle: セット販売のみ
├── Mixed Bundle: 単品もセットも選択可
└── Leader Bundle: 人気商品+在庫処分品

【価格設計】
├── 単品合計の70-85%をセット価格に
├── 「1点買うと2点目半額」
└── 「3点まとめ買いで20%OFF」
```

### 5. サブスクリプション価格
```
【基本原則】
月額/年額で支払い負担感を軽減

【設計パターン】
├── 月額: ¥980/月（低ハードル）
├── 年額: ¥9,800/年（2ヶ月分お得）
├── 生涯: ¥29,800（一度きり）

【日額換算表示】
├── 「1日たった¥33」（¥980/月）
├── 「コーヒー1杯以下」
└── 心理的負担感の軽減
```

## 損失回避の活用

### ロスフレーミング
```
【ゲイン（利得）フレーム】
「今なら10%OFF！」

【ロス（損失）フレーム】
「買わないと10%損します」
「明日から値上げ」
「この価格は今日まで」

【効果】
ロスフレームはゲインフレームより
1.5-2倍の行動喚起効果
```

### ポイント失効通知
```
【効果的な通知設計】
├── 30日前: 「○○ポイントが失効予定」（情報）
├── 7日前: 「あと7日で失効します」（警告）
├── 1日前: 「明日失効！今すぐ使う」（緊急）

【メッセージ例】
❌「1,000ポイントをお持ちです」
✅「1,000ポイントが3日後に失効します。¥1,000相当を失う前に」
```

## 価格アーキテクチャ

### 松竹梅（Good-Better-Best）
```
【基本構成】
├── 松（Premium）: 全機能、最高品質 ← 10%
├── 竹（Standard）: 主要機能、標準品質 ← 70%（狙い）
└── 梅（Basic）: 最低限機能、入門 ← 20%

【価格比率】
├── 梅: 1x
├── 竹: 1.5-2x
└── 松: 2-3x

【EC適用例】
├── 送料無料ライン: ¥5,000
├── プレミアム会員: ¥9,800/年
├── 通常会員: ¥4,980/年
└── 無料会員: ¥0
```

### フリーミアム
```
【基本構成】
├── 無料: 基本機能、広告あり
├── 有料: 全機能、広告なし ← 転換狙い

【転換率目安】
├── B2C: 2-5%
├── B2B: 5-15%

【設計ポイント】
├── 無料で価値を体験させる
├── 有料へのアップグレード動機を作る
├── 無料→有料の障壁を下げる
└── 一度有料を体験させる（トライアル）
```

## 成功事例

### Costco（会員制ロスリーダー）
- **モデル**: 年会費で利益、商品は原価販売
- **会員費**: $60-$120/年
- **更新率**: 92.6%
- **心理効果**: サンクコスト、コミットメント

### Amazon Prime
- **モデル**: 年会費でロックイン、追加購入促進
- **会員費**: ¥5,900/年（日本）
- **効果**: 会員は非会員の2倍購入
- **心理効果**: 送料無料で「使わないと損」

### Netflix
- **モデル**: 月額定額、プラン階層
- **価格**: ¥790-¥1,980/月
- **戦略**: 広告プラン導入で下位アンカー
- **心理効果**: 「広告なし」の価値を実感

```
【ポイント心理の活用】
├── ポイント付与: 「実質○%OFF」表現
├── ポイント倍率: お買い物マラソン活用

【価格表示例】
「¥3,980（実質¥3,582）※ポイント10%還元」
```

```
【価格設計】
├── 通常価格: ¥1,980
├── 定期便価格: ¥1,683（15%OFF）
├── まとめ割: さらに5%OFF

【心理効果】
├── 損失回避: 「定期便にしないと高い」
├── デフォルト: 定期継続がデフォルト
└── ロックイン: 解約の面倒さ
```

### Yahoo! PayPay連携
```
【施策設計】
├── PayPayポイント還元: 即時還元の魅力
├── クーポン: 「〇〇円OFF」の直接値引き
├── ボーナスストア: 追加ポイント

【価格表示】
「¥2,980 + PayPay 10%還元」
「実質¥2,682相当」
```

## 実装チェックリスト

### 商品ページ
```
□ アンカー価格（通常価格）の表示
□ 割引率または割引額の明示
□ 端数価格の適用（¥2,980等）
□ ポイント還元の可視化
□ 「あと○個」在庫表示
□ 「○人が閲覧中」表示
□ レビュー数・評価の表示
```

### カート・決済
```
□ 送料無料ラインまでの金額表示
□ 「あと○円で送料無料」表示
□ まとめ買い割引の提示
□ ポイント利用の選択肢
□ 分割払いオプション
□ 「よく一緒に購入される商品」
```

### 会員制度
```
□ ティア構造（松竹梅）
□ アップグレード特典の明示
□ 年会費の日額換算表示
□ 更新リマインダー設計
□ 失効ポイント通知
```

## KPI設計

| KPI | 計算式 | 目標値 |
|-----|--------|--------|
| 客単価 | 売上 / 注文数 | +10%（施策前比） |
| CVR | 購入者 / 訪問者 | +15%（施策前比） |
| カート追加率 | カート追加 / 訪問者 | +20% |
| まとめ買い率 | 2点以上購入 / 全購入 | 40%+ |
| 会員転換率 | 有料会員 / 全会員 | 10%+ |
| ポイント利用率 | ポイント利用購入 / 全購入 | 60%+ |
