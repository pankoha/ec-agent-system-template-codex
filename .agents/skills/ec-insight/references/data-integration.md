## コホート分析定義
```
時間軸コホート: 初回購入月別 / 登録月別 / 流入チャネル別
行動コホート: 購入回数別（F1/F2/F3+）/ 購入金額別 / 商品カテゴリ別
属性コホート: 年代別 / 地域別 / デバイス別

分析指標:
├── リテンション: 月次/90日/年間リピート率
├── LTV推移: 累積売上/購入回数/購入間隔
└── 行動変化: チャネル遷移/商品遷移/単価変化
```

## アトリビューションモデル比較
```
ラストクリック: シンプル、上流過小評価 → 単一チャネル時
線形: 均等評価 → 初期分析時
減衰: 直近重視 → 短期キャンペーン
位置ベース: 初回40%/中間20%/最終40% → 一般的なEC
データドリブン: ML最適化 → 成熟期
```
