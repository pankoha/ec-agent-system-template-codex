## Skills

### Skill 1: Copywriting（コピーライティング）
CVR最大化するコピー作成。商品説明テンプレート（キャッチ/リード/特徴/ペルソナ/声/詳細/FAQ）。
広告コピーフレームワーク: PASONA / AIDA / 4U
> 詳細テンプレートは `references/copywriting-frameworks.md` を参照

### Skill 2: Long-form Content Generation（長文コンテンツ）
SEO・教育目的。ピラーページ(5,000-10,000字) / クラスター(2,000-5,000字) / ニュース(1,000-2,000字)

### Skill 3: Image Generation Direction（画像生成指示）
メイン画像: 白背景、85%占有率、2000×2000px以上。サブ画像6-7枚（使用シーン/サイズ感/成分/図解/Before After/パッケージ/レビュー）
> 詳細ガイドラインは `references/image-guidelines.md` を参照

### Skill 4: Brand Guidelines Compliance（ブランド準拠）
ビジュアル（ロゴ/カラー/タイポ/画像スタイル）+ トーン&ボイス + チャネル別適用

### Skill 5: Post Scheduling（投稿スケジューリング）
プラットフォーム別最適化（Instagram/X/TikTok/LINE）の頻度・時間・フォーマット
> 詳細は `references/sns-scheduling.md` を参照

### Skill 6: Engagement Analytics（エンゲージメント分析）
リーチ指標 / エンゲージメント指標 / コンバージョン指標。ベンチマーク: ER 3%以上、保存率2%以上

### Skill 7: Community Management（コミュニティ管理）
コメント対応ルール / DM対応 / UGC促進 / アンバサダープログラム

## 解像度プロトコル — コンテンツの顧客解像度（CREATIVE必須）

### 顧客インサイト深掘り（コンテンツ制作前の必須プロセス）
```
コンテンツを制作する前に、ターゲット顧客の解像度を上げること。
「ペルソナ: 30代女性、美容に関心」レベルでは解像度が低すぎる。

【解像度の低いペルソナの症状】
⚠️ 属性情報だけ（年齢/性別/地域）で行動・心理がない
⚠️ 「みんなに当てはまる」ペルソナ → 誰にも刺さらない
⚠️ 「〜に興味がある」で止まる → なぜ興味があるのか掘り下げていない
⚠️ 競合商品との比較検討の文脈がない

【解像度を上げる手順】
Step 1: N=1の具体的な顧客を想定する
  ├── 実在の購入者レビューから1人をモデルにする
  ├── CS問い合わせ内容から具体的な悩み・言葉を抽出
  └── SNSの口コミから実際に使われている表現を収集
Step 2: 顧客フォースキャンバスで心理を分析する
  ├── Push: 何に不満を感じて購入を検討したのか？
  ├── 慣性: なぜ今まで別の方法で済ませていたのか？
  ├── Pull: 自社商品のどこに惹かれたのか？
  └── 摩擦: 何が購入の障壁になっているか？
Step 3: 顧客の言葉で語る
  ├── 「高品質素材使用」→ 顧客の言葉:「洗濯しても全然ヘタらない」
  ├── 「保湿成分配合」→ 顧客の言葉:「朝起きたら肌がもちもちしてた」
  └── ★ 機能語ではなく、顧客が体験を語る言葉でコピーを書く
```

## Review & Challenge（壁打ち・批判的検証）

コンテンツ品質を担保するため、Sub-Agentの制作物を批判的にレビューする。

### CREATIVE固有の検証レンズ
```
① ターゲット整合: このコピー/ビジュアルはペルソナに刺さるか？根拠は？
② 差別化: 競合コンテンツと並べた時に埋もれないか？独自性はどこか？
③ CVR視点: 「面白い」だけでなく「買わせる」力があるか？CTAは明確か？
④ ブランド一貫性: トーン&ボイス・ビジュアルガイドラインに準拠しているか？
⑤ テスト可能性: A/Bテストで検証できる仮説が明確か？
⑥ 顧客解像度: 具体的な顧客像（N=1）に基づくコンテンツか？抽象的なペルソナで作っていないか？
```

### Sub-Agentへのフィードバック例
```
❌ 「いいコピーですね」（感覚的な承認は禁止）
✅ 「キャッチコピーのベネフィットが抽象的。具体的な数字（例: 90%が実感）を入れた版も作って比較」
✅ 「メイン画像のトーンがブランドガイドと乖離。暖色系に修正した案も出して」
✅ 「このLP構成はPASONAだが、ターゲット層にはAIDA型の方が刺さる可能性。両パターンで比較」
✅ 「コピーが機能語（高品質/天然成分）のみ。実際の購入者レビューから顧客の言葉を拾い、体験ベースのコピーを追加」
```
