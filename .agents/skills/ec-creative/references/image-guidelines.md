# 画像生成ガイドライン

## メイン画像
```
├── 背景: 白（Amazon規約準拠）
├── 商品占有率: 85%以上
├── 解像度: 2000×2000px以上
├── 角度: 正面、やや上から
└── 影: 自然なドロップシャドウ
```

## サブ画像（6-7枚）
```
├── 使用シーン
├── サイズ感（手持ち、比較物）
├── 成分・原材料
├── 特徴図解
├── Before/After
├── パッケージ
└── レビュー/受賞歴
```

## バナー構成
```
├── メインビジュアル（左/中央）
├── キャッチコピー（大）
├── サブコピー（小）
├── CTA/オファー
├── ロゴ
└── 装飾（控えめに）
```

## ブランドガイドライン項目
```
ビジュアル:
├── ロゴ使用規定
├── カラーパレット（プライマリ/セカンダリ）
├── タイポグラフィ（フォント、サイズ）
├── 画像スタイル（明るさ、彩度）
└── アイコンスタイル

トーン&ボイス:
├── ブランドパーソナリティ
├── 使用OK/NG表現
├── 敬語レベル
├── 絵文字使用ルール
└── ハッシュタグ規定

チャネル別適用:
├── 自社EC: フォーマル、詳細
├── Amazon: 規約準拠、シンプル
├── 楽天: 販促感、イベント訴求
├── SNS: カジュアル、親しみやすさ
└── メール: パーソナル、温かみ
```
