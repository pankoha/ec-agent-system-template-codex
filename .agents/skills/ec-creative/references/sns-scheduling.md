# SNS投稿スケジューリング

## プラットフォーム別最適化

### Instagram
```
├── 頻度: 週3-5回（フィード）、日1-3回（ストーリー）
├── 時間: 12:00、19:00-21:00
├── フォーマット: カルーセル > リール > 単画像
├── ハッシュタグ: 10-15個（ビッグ2/ミドル7/スモール5）
└── CTA: プロフィールリンク誘導
```

### Twitter/X
```
├── 頻度: 日1-5回
├── 時間: 7:00、12:00、19:00
├── フォーマット: 画像付き > テキストのみ
├── ハッシュタグ: 2-3個
└── スレッド: 教育コンテンツに有効
```

### TikTok
```
├── 頻度: 週3-7回
├── 時間: 18:00-22:00
├── フォーマット: 縦型動画、15-60秒
├── フック: 最初の1秒が勝負
└── トレンド音源活用
```

### LINE公式
```
├── 頻度: 週1-2回
├── 時間: 12:00、20:00
├── フォーマット: リッチメッセージ、カードタイプ
├── 長さ: 簡潔に（3メッセージ以内）
└── CTA: タップで遷移
```

## KPIベンチマーク
```
リーチ指標: インプレッション / リーチ / フォロワー増加率 / シェア数
エンゲージメント指標: ER 3%以上（良好）/ 保存率 2%以上 / コメント率
コンバージョン指標: リンクCTR 1%以上 / プロフィール訪問率 / SNS経由売上
フォロワー増加: 週2%以上
```

## コミュニティ管理
```
コメント対応:
├── ポジティブ: いいね + お礼返信
├── 質問: 24時間以内に回答
├── ネガティブ: 丁寧に対応、DMへ誘導
├── スパム: 非表示 + ブロック
└── UGC: リポスト許可取得 + 紹介

コミュニティ育成:
├── UGC促進ハッシュタグ
├── フォロワー参加型企画
├── ライブ配信（月1回）
└── アンバサダープログラム
```
