# コピーライティングフレームワーク

## 商品説明テンプレート
```markdown
【キャッチコピー】
[数字] + [ベネフィット] + [差別化]
例: 累計10万本突破！糖質制限中でも罪悪感なく楽しめる

【リード文】（100-150文字）
[ターゲットの悩み]を解決する[商品]。
[主要ベネフィット]で[具体的な結果]を実現。[証拠・権威性]。

【特徴】（3-5つ）
✓ [特徴1]: [ベネフィット]
✓ [特徴2]: [ベネフィット]
✓ [特徴3]: [ベネフィット]

【こんな方におすすめ】
・[ペルソナ1の悩み]を解決したい方
・[ペルソナ2の願望]を叶えたい方

【お客様の声】
「[具体的な体験談]」（[属性]）

【商品詳細】
内容量: / 原材料: / 賞味期限: / 保存方法:

【よくある質問】
Q: [質問1]
A: [回答1]
```

## 広告コピーフレームワーク

### PASONA
```
P - Problem: 問題提起
A - Affinity: 共感
S - Solution: 解決策
O - Offer: 提案
N - Narrow: 限定
A - Action: 行動喚起
```

### AIDA
```
A - Attention: 注意喚起
I - Interest: 興味喚起
D - Desire: 欲求喚起
A - Action: 行動喚起
```

### 4U
```
U - Useful: 有用性
U - Urgent: 緊急性
U - Unique: 独自性
U - Ultra-specific: 具体性
```

## ブログ記事構成
```
1. タイトル（32文字以内、KW含む）
2. リード文（200文字、フック）
3. 目次
4. H2: 問題提起
5. H2: 解決策概要
6. H2: 詳細解説1（H3サブトピック）
7. H2: 詳細解説2
8. H2: 実践方法・手順
9. H2: よくある質問
10. まとめ + CTA

文字数: ピラー5,000-10,000字 / クラスター2,000-5,000字 / ニュース1,000-2,000字
```
