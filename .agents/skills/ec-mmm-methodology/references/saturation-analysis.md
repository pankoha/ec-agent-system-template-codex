# 飽和曲線分析の実践ガイド

## 飽和曲線の読み方マスターガイド

### 飽和曲線のビジュアル解釈

```
売上
 ▲
 │                         ┌──── 理論最大値（max_revenue）
 │                    ....─┘
 │               ...─
 │          ...─       ← 飽和ゾーン（80%+）: 追加投資の効果が急激に低下
 │        .─
 │      .─             ← 効率逓減ゾーン（50-80%）: まだ伸びるが効率は低下
 │    .─
 │   .─                ← 成長ゾーン（20-50%）: 投資効率が最も高い
 │  /
 │ /                   ← 立ち上がりゾーン（0-20%）: 最小投資で効果検証
 │/
 └─────────────────────────────────▶ 支出
       K（半飽和点）
```

### 飽和度別の詳細アクションプラン

#### 0-20%: 立ち上がりゾーン
```
状態: チャネルの潜在力をほとんど使っていない
意味: 投資余地が非常に大きいが、データも少ない

アクション:
├── まずは小規模テスト（2-4週間）で反応を確認
├── Hill R²が低い可能性が高い → 飽和曲線の形状は参考程度
├── 限界ROIの数値よりも、テスト結果の定性的な評価を重視
└── 2倍ずつ段階的に増額し、効果測定を繰り返す

リスク:
├── データ不足でモデルの信頼性が低い
└── 飽和曲線の形状が変わる可能性がある
```

#### 20-50%: 成長ゾーン（最も重要）
```
状態: 投資効率が最も高いスイートスポット
意味: 追加1円あたりのリターンが最大

アクション:
├── 優先的に予算を配分する最有力候補
├── 限界ROIを週次でモニタリング
├── 競合の動向によって飽和点が変動するため定期的に再分析
└── 月次で10-30%ずつ段階的に増額

目標: このゾーンにあるチャネルを見つけて集中投資する
```

#### 50-80%: 効率逓減ゾーン
```
状態: 成熟したチャネル。まだ伸びるが効率は低下中
意味: 投資は意味があるが、ROIは下がり始めている

アクション:
├── 現状維持 or 微増（月5-10%程度）
├── 限界ROIが目標値を下回ったら増額を停止
├── クリエイティブリフレッシュで飽和を遅延させる
├── オーディエンス拡張で新しい層にリーチ
└── 他チャネルとのシナジーで効果を補強

チェックポイント:
├── 限界ROI > 1.5 → まだ増額余地あり
├── 限界ROI 1.0-1.5 → 要注意。他チャネルとの比較で判断
└── 限界ROI < 1.0 → 削減してでも他チャネルに回すべき
```

#### 80-95%: 飽和ゾーン
```
状態: チャネルの効果がほぼ頭打ち
意味: 追加投資のROIが非常に低い

アクション:
├── 積極的な削減を検討（10-20%削減テスト）
├── 削減した予算を成長ゾーンのチャネルに再配分
├── ブランド認知目的なら維持も選択肢（ただし効率は低い）
├── 飽和の根本原因を調査:
│   ├── オーディエンスの枯渇？ → 新規ターゲット開拓
│   ├── クリエイティブ疲弊？ → 全面刷新
│   └── 季節要因？ → 時期をずらして再評価
└── 削減テスト実施 → 売上影響を計測 → MMM再実行

注意: 飽和=不要ではない。維持費として必要な投資額を見極める
```

#### 95%+: 過飽和ゾーン
```
状態: これ以上の投資は実質的に無駄
意味: 追加投資のリターンがほぼゼロ

アクション:
├── 即座に20-30%の予算削減を実施
├── 売上への影響が限定的であることを確認（飽和しているなら影響小）
├── 浮いた予算を飽和度の低いチャネルに投下
├── 3-4週間後にMMM再実行で効果を検証
└── 段階的に適正水準まで削減

警告:
├── 過飽和チャネルの予算を一気にゼロにしない（ベースライン効果がある）
├── シナジーのあるチャネルへの波及影響を確認してから削減
└── 競合が同チャネルで積極投資している場合、完全撤退はリスク
```

## 飽和度の経時変化を追跡する

```
飽和度トレンドが重要な理由:
単一時点の飽和度だけでは不十分。時系列で追うことで構造変化を検知できる。

追跡テンプレート:
| 月 | チャネル | 飽和度 | 限界ROI | 投下額 | アクション |
|----|---------|--------|---------|--------|----------|
| 1月 | Meta | 55% | 2.1x | ¥1,500万 | 現状維持 |
| 2月 | Meta | 62% | 1.8x | ¥1,500万 | 微増 |
| 3月 | Meta | 71% | 1.3x | ¥1,700万 | クリエイティブ刷新 |
| 4月 | Meta | 65% | 1.9x | ¥1,700万 | 刷新効果あり！ |

注目パターン:
├── 飽和度が月5%+上昇 → クリエイティブ疲弊の可能性が高い
├── 投下額据え置きなのに飽和度上昇 → 市場の変化（競合参入等）
├── 飽和度が下がった → クリエイティブ刷新/新オーディエンス獲得の効果
└── 全チャネル同時に飽和度上昇 → 季節要因の可能性
```

## Hill R²が低い場合の対処法

```
Hill R² < 0.1 の場合:

原因と対処:
├── データ量不足（30日未満）
│   → 最低60日、理想は180日分のデータを蓄積
│
├── 支出パターンが均一（毎日同額）
│   → 意図的に予算変動を作る「支出バリエーションテスト」を実施
│   → 例: 1週間を「高・中・低」の3パターンで運用
│
├── 外れ値の影響
│   → セール日など突発的な高額投下日を特定し、外部要因として登録
│
└── チャネル特性がHill関数に合わない
    → 線形に近い応答（S≈1）の場合、Hill R²は低くても飽和分析は参考にできる
    → 閾値効果がある場合（一定額以上で急に効果が出る）、別のモデルが必要

判断基準:
├── Hill R² > 0.5 → 飽和曲線を信頼してよい
├── Hill R² 0.1-0.5 → 大まかな傾向として活用（数値の正確性は低い）
└── Hill R² < 0.1 → 飽和曲線の数値は使わない。平均ROI/限界ROIで判断
```
