# 品質チェックリスト

## 使い方

画像構成案やプロンプトを提出する前に、このチェックを通す。問題がある場合は「要確認」「素材不足」「表現リスク」として明示し、完成案に混ぜ込まない。

## Amazon基本チェック

- メイン画像は白背景か
- メイン画像に販売商品以外が入っていないか
- メイン画像に文字、バッジ、枠、色付きブロック、透かし、背景装飾がないか
- メイン画像の商品は十分に大きく、全体が見えるか
- サブ画像とメイン画像のルールを混同していないか
- 画像が商品を正確に表しているか
- 同梱物、容量、数量、サイズに誤認がないか

## 購買導線チェック

- 1枚目で商品が何か一瞬でわかるか
- 2枚目以降に役割の重複が少ないか
- 悩み、機能、使用シーン、比較、サイズ、信頼が順に解消されるか
- 画像ごとに主役の訴求が1つに絞られているか
- スマホ表示で見出しが読めるか
- 文字が多すぎないか
- 商品写真が小さすぎないか
- 最後に購入理由が再確認できるか

## デザイン品質チェック

- 商品の輪郭、質感、光沢、影が高品質に見えるか
- ジャンルに合う配色か
- 画像セット全体でトーン、フォント、余白、強調色が揃っているか
- バッジ、矢印、吹き出し、比較表が視線誘導に効いているか
- 背景が商品より目立っていないか
- 使用シーンがターゲットの日常と一致しているか
- 安っぽい素材合成や不自然な手・顔・影がないか

## コピー品質チェック

- 機能ではなく顧客ベネフィットに翻訳されているか
- メインコピーは短く強いか
- サブコピーは説明しすぎていないか
- 顧客が使う言葉に近いか
- 競合より選ぶ理由が明確か
- 断定表現に根拠があるか

## 法務・規約リスクチェック

- No.1、最高、最安、唯一、永久、完全などに根拠があるか
- 医療効果、治療、予防、改善、痩身、美白、育毛などの表現を安易に使っていないか
- 食品・サプリで疾病名や身体機能改善を断定していないか
- 美容商品で薬機法に抵触しそうなBefore/Afterを使っていないか
- 比較表現に根拠があるか
- レビュー、受賞、販売実績、満足度などの数値に根拠があるか
- 著名人、芸能人、特定ブランドに似せたモデル・意匠を要求していないか
- 子供、医療、健康、安全に関わる表現で危険な使い方を示していないか

## 画像生成AI特有チェック

- 商品形状が実物と変わっていないか
- ロゴ、文字、ラベルが崩れていないか
- 手指、顔、身体、反射、影が不自然でないか
- 商品と人物の接触が不自然でないか
- 商品に含まれない付属品が生成されていないか
- 色違い・サイズ違い・セット数違いが混ざっていないか
- 文字入れ前提の安全余白があるか

## 提出時の確認事項テンプレート

```markdown
## 要確認

- [ ] 商品の正確なサイズ・重量
- [ ] 同梱物とSKUごとの差分
- [ ] 使用可能な実績・レビュー・受賞歴
- [ ] 競合比較に使える根拠
- [ ] 薬機法・景表法チェックが必要な表現
- [ ] 使用可能な写真素材、ロゴ、ブランドカラー
```
