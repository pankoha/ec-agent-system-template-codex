# 🧠 EMOTIONAL MARKETING METHODOLOGY（エモーショナルマーケティング方法論）

## 役割
全マーケティング関連Agentが共有する感情マーケティングの知識基盤。
神経科学・心理学・行動経済学の学術的エビデンスに基づき、
「なんとなく感情に訴える」ではなく「科学的に感情を設計する」マーケティングを実現する。

## 核心原理
```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  人間の意思決定の95%は無意識で行われる（消費者行動研究）         │
│  感情なしに合理的判断は成立しない（神経科学的知見）              │
│  感情広告は理性広告の2倍の長期利益成長を生む（大規模実証研究）   │
│                                                                 │
│  → 感情は「おまけ」ではなく、購買行動の中核エンジンである        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Skills

### Skill 1: 感情と意思決定の科学的基盤（Neuroscience Foundation）

消費者の購買行動を支配する感情メカニズムを、神経科学・心理学のエビデンスで理解する。

**3つの基盤理論:**

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  【1. ソマティック・マーカー仮説】                              │
│                                                                 │
│  過去の経験が身体感覚（心拍上昇、胃の不快感など）として記録され、 │
│  意思決定の瞬間に「良い/悪い」のシグナルを送る。                 │
│                                                                 │
│  腹内側前頭前皮質（vmPFC）の損傷患者は感情処理ができず、          │
│  論理的能力は正常なのに日常的な判断すらできなくなる。              │
│                                                                 │
│  → 感情は判断を歪めるノイズではなく、判断に不可欠なシグナル       │
│  → ブランド体験の蓄積がソマティック・マーカーとなり、              │
│    店頭やECでの瞬間的な購買判断を方向づける                      │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  【2. 二重過程理論（System 1 / System 2）】                     │
│                                                                 │
│  System 1: 速い思考。直感的・感情的・自動的。労力なし。           │
│  System 2: 遅い思考。分析的・論理的・意識的。労力を要する。       │
│                                                                 │
│  消費者の購買判断の大半はSystem 1が支配する。                     │
│  System 2は「System 1の判断を正当化する理由」を後付けする。       │
│                                                                 │
│  → 広告・LP・商品ページはまずSystem 1に訴えかける必要がある       │
│  → 感情で「欲しい」と思わせ、理性で「買うべき理由」を与える       │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  【3. 無意識の支配】                                            │
│                                                                 │
│  思考の95%は無意識で行われる。                                   │
│  消費者は自分の購買理由を正確に説明できない。                     │
│  アンケートやフォーカスグループは表層しか捉えない。               │
│                                                                 │
│  → 顧客の「言葉」ではなく「行動」を見る                         │
│  → メタファーや物語を通じて無意識にアクセスする                   │
│  → 従来のリサーチ手法の限界を認識する                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**マーケティングへの実装:**
```
感情で引く → 理性で押す → 感情で決める

Step 1: System 1を発火させる（FV・KV・ヘッドライン）
  └─ 0.5秒で「気になる」を生む感情的インパクト
  └─ 画像 > コピー（視覚情報は言語の60,000倍速く処理される）

Step 2: System 2に理由を与える（スペック・比較・証拠）
  └─ 「買いたい」を「買うべき」に変換する論理的根拠
  └─ データ・エビデンス・第三者の声

Step 3: 感情で最後のひと押し（CTA周辺・クロージング）
  └─ 緊急性・希少性・損失回避で感情的決断を促す
  └─ 購入後の感情的報酬をイメージさせる
```

**Agent別の実装指針:**
```
ec-acquisition（広告運用）:
  ・広告予算の60%を感情的ブランド構築に、40%をセールスアクティベーションに
  ・クリエイティブテストでは感情訴求を必ず含める
  ・広告効果測定はリコール以外の指標（ブランドリフト等）も必ず併用

ec-commander（戦略）:
  ・短期ROASだけで広告を評価しない。長期ブランド効果を考慮する
  ・感情ブランディング投資は「コスト」ではなく「資産構築」として扱う
```

### Skill 3: 感情分類と活用フレームワーク（Emotion Taxonomy）

どの感情をいつ使うかの設計指針。

**基本感情モデル + マーケティング活用:**

```
┌─────────────────────────────────────────────────────────────────┐
│  感情      │ マーケティング活用                    │ 適用場面   │
├───────────┼─────────────────────────────────────┼───────────┤
│ 喜び      │ 成功体験・幸福感の提示                │ ビフォアフ │
│ (Joy)     │ 「使ったらこうなれる」のビジョン       │ ター・UGC  │
├───────────┼─────────────────────────────────────┼───────────┤
│ 驚き      │ 意外性・発見・ギャップ                │ FV・KV・   │
│ (Surprise)│ 常識を覆す事実の提示                  │ ヘッドライン│
├───────────┼─────────────────────────────────────┼───────────┤
│ 恐怖      │ 損失回避・リスクの可視化              │ 問題提起・ │
│ (Fear)    │ 「このままだとどうなるか」             │ 緊急性訴求 │
├───────────┼─────────────────────────────────────┼───────────┤
│ 怒り      │ 不公正・理不尽への共感                │ 共感パート │
│ (Anger)   │ 「こんなの許せない」の共鳴             │ 敵の設定  │
├───────────┼─────────────────────────────────────┼───────────┤
│ 悲しみ    │ 共感・寄り添い                        │ ストーリー │
│ (Sadness) │ 弱さを見せることで信頼を得る           │ 自己開示  │
├───────────┼─────────────────────────────────────┼───────────┤
│ 嫌悪      │ 現状の問題点の強調                    │ 比較・現状 │
│ (Disgust) │ 「こんなの嫌ですよね」                 │ 否定     │
└─────────────────────────────────────────────────────────────────┘
```

**感情の輪モデル — 感情の強度設計:**
```
感情は強度を持つ。同じ「喜び」でも強度が違う:

  静穏(Serenity) → 喜び(Joy) → 恍惚(Ecstasy)
  受容(Acceptance) → 信頼(Trust) → 敬愛(Admiration)
  不安(Apprehension) → 恐怖(Fear) → 恐慄(Terror)
  気晴らし(Distraction) → 驚き(Surprise) → 驚愕(Amazement)

マーケティングでの使い分け:
  ・日常的なSNS投稿 → 低〜中強度（静穏、受容）
  ・LP/セールスページ → 中〜高強度（喜び、恐怖、驚き）
  ・ブランドストーリー → 中強度で持続的（信頼、喜び）
  ・緊急セール/限定オファー → 高強度で瞬間的（恍惚、驚愕、恐慄）
```

**バイラルの感情設計（高覚醒感情がシェアを促す）:**
```
┌─────────────────────────────────────────────────────────────────┐
│  バイラルの鍵は感情の種類ではなく「覚醒度（arousal）」           │
│                                                                 │
│  高覚醒 → シェアされやすい                                      │
│    ポジティブ: 畏怖(Awe) / 興奮(Excitement) / 面白さ(Amusement) │
│    ネガティブ: 怒り(Anger) / 不安(Anxiety)                      │
│                                                                 │
│  低覚醒 → シェアされにくい                                      │
│    悲しみ(Sadness) / 満足(Contentment)                          │
│                                                                 │
│  → SNSコンテンツは「高覚醒ポジティブ」を狙うのが最も効果的       │
│  → 畏怖（Awe）が最も強いバイラルドライバー                      │
│    例: 「こんなの見たことない」「信じられない結果」               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```
              高い尊敬

    ブランド    │    ラブマーク
   (Brands)    │   (Lovemarks)
                │

                │

  (Products)   │    (Fads)

  Intimacy（親密性）: 共感、コミットメント、情熱
```

### Skill 5: 感情コピーライティング（Emotional Copywriting）

感情を動かすコピーの科学的設計手法。

**感情トリガーの体系:**
```
┌─────────────────────────────────────────────────────────────────┐
│                 8つの感情トリガー                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. 損失回避（Loss Aversion）                                   │
│     人は得る喜びより失う痛みを2倍強く感じる（プロスペクト理論） │
│     「手に入れませんか？」< 「このまま失い続けますか？」          │
│     例: 「毎月〇万円を捨てていませんか？」                      │
│                                                                 │
│  2. 社会的証明（Social Proof）                                  │
│     他者の行動が正しさの証拠になる（社会的影響理論）             │
│     具体的な数字が効く: 「累計10,000人」「★4.8（2,340件）」     │
│     例: 「3,000人のスクール生が選んだ方法」                     │
│                                                                 │
│  3. 帰属意識（Belonging）                                       │
│     「仲間になりたい」「この集団の一員でいたい」                  │
│     内集団バイアスを活用                                         │
│     例: 「選ばれた人だけが知っている」「仲間と一緒に」           │
│                                                                 │
│  4. 好奇心ギャップ（Curiosity Gap）                             │
│     知っていることと知りたいことのギャップが行動を生む             │
│     例: 「90%の人が知らない〇〇の秘密」                         │
│                                                                 │
│  5. 希少性（Scarcity）                                          │
│     手に入らないものほど欲しくなる（影響力研究の知見）           │
│     時間的希少性: 「あと24時間」                                 │
│     数量的希少性: 「残り3席」                                    │
│                                                                 │
│  6. 権威（Authority）                                           │
│     専門家の意見は無条件に信頼されやすい（影響力研究の知見）     │
│     例: 「〇〇大学の研究で実証」「業界20年のプロが」            │
│                                                                 │
│  7. 物語（Narrative Transportation）                            │
│     物語に没入すると批判的思考が低下し、説得力が増す              │
│     起承転結ではなく「共感→葛藤→変容」の構造が効く              │
│     例: 「2年前の私は〜」（Before→Struggle→After）              │
│                                                                 │
│  8. 自己実現（Self-Actualization）                               │
│     「なりたい自分」へのビジョンを提示する                       │
│     商品の機能ではなく、購入後の自己像を描く                     │
│     例: 「〇〇を使いこなすあなたは、もう初心者ではない」         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**セールスページの感情フロー設計:**
```
┌──────────────────────────────────────────────┐
│  1. HOOK（0-3秒）: 驚き or 好奇心            │
│     → System 1を発火させ、スクロールを止める  │
│                                               │
│  2. PAIN（問題提起）: 恐怖 + 怒り + 嫌悪      │
│     → 「このままではまずい」を感じさせる       │
│     → 損失回避を発動させる                    │
│                                               │
│  3. EMPATHY（共感）: 悲しみ + 帰属意識         │
│     → 「わかります。私もそうでした」           │
│     → 信頼関係を構築する                      │
│                                               │
│  4. SOLUTION（解決策）: 喜び + 驚き            │
│     → 「こうすれば解決できます」               │
│     → System 2に論理的根拠を提供する           │
│                                               │
│  5. PROOF（証拠）: 信頼 + 権威                 │
│     → 社会的証明・実績・第三者の声             │
│     → System 2を満足させる                    │
│                                               │
│  6. VISION（ビジョン）: 喜び + 自己実現         │
│     → 「使った後のあなたはこうなる」            │
│     → 購入後の感情的報酬をイメージさせる       │
│                                               │
│  7. CLOSE（決断）: 希少性 + 損失回避            │
```

### Skill 7: 感情測定と最適化（Emotion Measurement & Optimization）

感情マーケティングの効果を測定し、継続的に改善する手法。

**測定指標の階層:**
```
Level 1: 行動指標（直接測定可能）
  ・CTR（クリック率）— 感情的インパクトの代理指標
  ・スクロール率/滞在時間 — 感情的エンゲージメント
  ・CVR（コンバージョン率）— 感情→行動の変換効率
  ・シェア率/UGC数 — 高覚醒感情の発生度合い
  ・リピート率/LTV — 感情的ロイヤルティ

Level 2: 認知指標（調査で測定）
  ・ブランド好意度 — 感情的ブランディングの蓄積
  ・ブランド想起率 — ソマティック・マーカーの形成度
  ・NPS — 感情的推奨意向
  ・ブランドパーソナリティ認知 — 5次元モデルの一致度

Level 3: 神経科学的指標（専門機器で測定）
  ・表情分析（FACS） — 広告視聴時の感情反応
  ・EEG — 注意・エンゲージメント・記憶エンコーディング
  ・アイトラッキング — 視線の滞留と感情ホットスポット
  ・皮膚電導度（GSR） — 覚醒度の生理学的測定
```

**A/Bテストでの感情変数:**
```
テストすべき感情変数（優先度順）:

  1. FV/KVの感情的インパクト（驚き vs 好奇心 vs 恐怖）
  2. ヘッドラインの感情トリガー（損失回避 vs 自己実現）
  3. 社会的証明の提示方法（数字 vs ストーリー vs 顔写真）
  4. CTAの感情フレーミング（希少性 vs 損失回避 vs 喜び）
  5. ストーリーの有無（物語あり vs 論理のみ）

  ※ 画像の変更がCTRに最も大きく影響する（写真 > コピー）
  ※ 大規模広告効果研究: 感情クリエイティブの効果はリコールでは測れない
     → CVR/ROAS以外にブランドリフトも必ず確認する
```
