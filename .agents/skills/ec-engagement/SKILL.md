5つの主要シナリオ:
1. ウェルカム（即時→3日→7日→14日、目標: 初回購入率30%）

## 解像度プロトコル — カスタマーマニア（ENGAGEMENT必須）

### N=1顧客深掘り分析（Customer Deep Dive）
```
セグメント統計だけでなく、個別の顧客を深く理解するN=1分析を必ず実施すること。

【N=1分析の実施タイミング】
├── 新しい施策を設計するとき（最低1人のターゲット顧客を深掘り）
├── CVRや開封率が目標未達のとき（CV顧客1人 vs 非CV顧客1人を比較）
├── 新商品ローンチ前（想定顧客1人の購買行動をストーリー化）
└── 四半期ごとの定期レビュー（各セグメントの代表顧客1人ずつ更新）

【N=1分析の手順】
Step 1: 特定の1人を選ぶ（最もそのセグメントを象徴する顧客）
Step 2: その顧客の全行動履歴を時系列で追う
  ├── 初回接触〜初回購入〜リピート（または離脱）の全ステップ
  ├── 各ステップでの滞在時間・クリック・閲覧ページ
  └── CS問い合わせ内容、レビュー、メール開封パターン
Step 3: 行動の「理由」を推察する（Why So?を3回以上）
  ├── なぜこのタイミングで購入したのか？
  ├── なぜこの商品を選んだのか？（他の選択肢は何だったか？）
  └── なぜリピートした（しなかった）のか？
Step 4: ストーリーとして記述する
  「〇〇さん（30代女性）は、Instagramの広告で商品を知り、
   LPで3分間熟読した後、一度離脱。翌日メールのカート離脱リマインドで
   戻り、レビューを5件読んでから購入。決め手は…」
Step 5: このストーリーから施策仮説を導出する
```

### 顧客フォースキャンバス（Customer Forces Canvas）
```
顧客の行動変化を駆動する4つの力を分析する:

1. Push（前進の動機）: 既存の状況に対する不満。何が顧客を動かしたか？
   └── イベントトリガー / 危機 / 不便・不満の蓄積

2. 慣性（既存の方法の快適さ）: 変えたくない理由。何が留まらせているか？
   └── 古い習慣 / スイッチングコスト / 「まあいいか」の心理

3. Pull（新しい方法の魅力）: 自社商品に引き寄せる力。何が魅力的か？
   └── ユニークな価値提案 / 望んでいるアウトカム / 想像する理想の未来

4. 摩擦（新しい方法への抵抗）: 購入を妨げる力。何が不安にさせているか？
   └── 不安・不確実性 / 新しい習慣への抵抗 / 価格の躊躇

★ 施策設計時に4つの力のバランスを必ず確認する
★ Push + Pull > 慣性 + 摩擦 のときに顧客は行動する
★ 施策が4つの力のどこに作用するかを明示すること
```

### 顧客の言葉の裏を読む（顧客の医師プロトコル）
```
顧客が言うこと = 症状（表面的な訴え）
本当の課題 = 病因（起業家/事業者が解明すべき真の原因）

❌ 禁止: 顧客の要望をそのまま施策に反映する
  「もっと安くしてほしい」→ 値下げする
✅ 必須: 顧客の言葉の裏を探る
  「もっと安くしてほしい」→ Why So? →
  ├── 価格に見合う価値を感じていない → 価値訴求の改善
  ├── 比較対象が安価な代替品 → ポジショニングの見直し
  └── 実は価格ではなく送料が問題 → 送料無料ラインの最適化
```

## Review & Challenge（壁打ち・批判的検証）

顧客施策の品質を担保するため、Sub-Agentの提案を批判的に検証する。

### ENGAGEMENT固有の検証レンズ
```
① セグメント精度: そのセグメント定義は実態と合っているか？過去の反応率で裏付けがあるか？
② 配信疲弊: 同じ顧客に過剰配信していないか？配信頻度と解約率の関係は確認したか？
③ パーソナライズ深度: 「名前を入れただけ」ではなく、行動ベースの真のパーソナライズか？
④ LTV貢献: この施策は短期売上だけでなく、長期LTV向上に寄与するか？
⑤ CS品質: テンプレ対応で顧客の本質的な不満を見逃していないか？
⑥ 顧客解像度: N=1の具体的な顧客ストーリーに基づく施策設計か？セグメント数字だけで判断していないか？
```

### Sub-Agentへのフィードバック例
```
❌ 「配信しましょう」（検証なき実行は禁止）
✅ 「カート離脱メールの回収率が8%で目標15%未達。件名のA/Bテスト結果と、クーポン額の比較データを出して」
✅ 「VIPセグメントの定義がRFM 555のみ。購入頻度が高いが単価の低い顧客（F5M2）も含めるべきか検討して」
✅ 「ウェルカムシリーズの3通目で離脱が急増。内容とタイミングの両面で改善案を2パターン出して」
```
