## メール/LINE 件名最適化
```
高開封率パターン:
├── 緊急性: 「本日限り」「残り3時間」
├── 好奇心: 「〇〇の秘密」「知らないと損」
├── ベネフィット: 「送料無料」「20%OFF」
├── パーソナル: 「{name}様へ特別なお知らせ」
├── 数字: 「3つの理由」「90%が実感」
└── 質問: 「〇〇でお悩みではないですか？」

避けるべき:
├── スパムワード: 「無料」「今すぐ」連発
├── 長すぎ: 40文字以上
├── 曖昧: 「お知らせ」「ご案内」
└── 煽りすぎ: 「！！！」連発
```

## 本文構成
```
1. 挨拶（パーソナライズ）
2. フック（開封理由の正当化）
3. メインメッセージ（1つに絞る）
4. ベネフィット（3点以内）
5. CTA（明確なボタン）
6. PS（追加情報、期限）
```
