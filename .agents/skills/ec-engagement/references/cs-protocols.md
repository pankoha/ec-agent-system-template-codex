# カスタマーサポートプロトコル

## 問い合わせ意図分類

### カテゴリ
```
商品関連: 商品仕様の質問 / 在庫確認 / 使い方 / 比較・選び方
注文関連: 注文状況確認 / 配送状況 / 注文変更 / キャンセル依頼
返品・交換: 返品依頼 / 交換依頼 / 返金確認 / 不良品報告
その他: 会員登録 / ポイント / クーポン / その他
```

### 対応優先度
```
🔴 最優先（2時間以内）:
├── クレーム・苦情
├── 不良品・破損
├── 誤配送
└── SNS投稿示唆

🟡 優先（24時間以内）:
├── 返品・返金
├── 注文変更
├── 配送遅延
└── 商品問い合わせ

🟢 通常（48時間以内）:
├── 一般質問
├── 要望・提案
├── 会員登録
└── その他
```

## 感情分析と対応

### 感情分類
```
ポジティブ:
├── 満足、喜び、感謝
├── 対応: お礼、レビュー依頼、紹介促進
└── 優先度: 低（良好な関係維持）

ニュートラル:
├── 質問、確認、要望
├── 対応: 丁寧な回答、情報提供
└── 優先度: 中

ネガティブ:
├── 不満、怒り、失望
├── 対応: 謝罪、迅速解決、フォローアップ
└── 優先度: 高

エスカレーション:
├── 法的示唆
├── SNS/メディア言及
├── 繰り返しクレーム
└── 対応: 即時上長対応
```
