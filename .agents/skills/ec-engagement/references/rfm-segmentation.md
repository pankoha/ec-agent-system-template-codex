# RFMセグメンテーション 詳細

## RFMスコアリング

### R（Recency - 最終購入日）
```
5: 30日以内
4: 31-60日
3: 61-90日
2: 91-180日
1: 181日以上
```

### F（Frequency - 購入回数）
```
5: 10回以上
4: 6-9回
3: 3-5回
2: 2回
1: 1回
```

### M（Monetary - 購入金額）
```
5: ¥50,000以上
4: ¥30,000-49,999
3: ¥15,000-29,999
2: ¥5,000-14,999
1: ¥5,000未満
```

## パーソナライゼーション要素
```
コンテンツ: 商品レコメンド/カテゴリ/価格帯/タイミング
メッセージ: 件名(名前,購買商品)/本文(セグメント別)/オファー(顧客価値別)/CTA(行動履歴別)
チャネル: メール優先(開封率高い)/LINE優先(メール開封低い)/SMS(緊急,高LTV)/アプリPush
```

## カスタマージャーニーステージ
```
認知 → 興味 → 検討 → 購入 → 体験 → 推奨

各ステージ:
├── タッチポイント
├── KPI
└── 施策
```
