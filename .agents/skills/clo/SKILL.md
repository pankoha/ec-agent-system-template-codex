### 日本における関連判例・法的根拠
```
1. 食べログ判例（2021年）
   ├── 事案: 口コミサイトからの大量スクレイピング
   ├── 判決: 不正競争防止法違反（営業秘密の不正取得）
   └── 教訓: 継続的な大量データ収集は違法リスク

2. 著作権法（データベースの著作物）
   ├── 条文: 著作権法12条の2
   ├── 適用: 創作性のあるデータベースは著作物として保護
   └── 教訓: ECサイトの商品DBは保護対象の可能性

3. 不正競争防止法（営業秘密）
   ├── 条文: 不正競争防止法2条1項4号
   ├── 適用: 秘密管理性・有用性・非公知性を満たす情報
   └── 教訓: 価格戦略・顧客データ等は営業秘密の可能性

4. 電子計算機損壊等業務妨害罪
   ├── 条文: 刑法234条の2
   ├── 適用: サーバーに過度な負荷をかける行為
   └── 教訓: DoS的なスクレイピングは刑事罰リスク
```

## Review & Challenge（壁打ち・批判的検証）

### CLO固有の検証レンズ
```
① リスク評価精度: 法的リスクの発生確率と影響度は定量化されているか？
② 規制変更対応: 最新の法改正（景品表示法/薬機法/個人情報保護法/各国PDPA）は反映されているか？
③ 先例確認: 類似ケースの判例・行政処分事例は調査されているか？
④ 予防 vs 対応: 事後対応だけでなく、予防的な法務レビューが組み込まれているか？
⑤ ビジネスインパクト: 法的リスク回避がビジネス機会の過度な制約になっていないか？
⑥ 越境対応: 各国法規制への対応は網羅されているか？データ越境移転は適法か？
⑦ プラットフォーム規約: 利用プラットフォームの規約は遵守されているか？
```

### 越境法務フィードバック例
```
❌ 「スクレイピングで競合データを収集しましょう」（違法リスク）
✅ 「Makuakeのrobots.txtとToSを確認。GPTBotはDisallow。代替としてユーザーアップロード方式を推奨」

❌ 「海外に顧客データを送信しても大丈夫です」（確認なし）
✅ 「台湾への顧客データ移転には個人資料保護法の要件確認が必要。同意取得と安全管理措置を設計して」

❌ 「法的にグレーなので避けましょう」（根拠なき過度な保守は禁止）
✅ 「広告表現のリスク評価だが、措置命令の類似事例を3件以上調査して判断根拠を強化して」
✅ 「契約書のリスク条項指摘に対し、ビジネス上の代替案も併せて提案して」
```

```
│  ├── 「顧客満足度No.1」の調査根拠が不適切または存在しない       │
│  ├── 「業界最安値」を根拠なく表示                               │
│  ├── 「売上No.1」の集計期間・対象範囲が不明確                   │
│  ├── 加工写真で実物と著しく異なる印象を与える                   │
│  └── 原産地・原材料の虚偽表示                                   │
│                                                                 │
│  【ステマ規制（2023年10月施行）】                               │
│  ├── 事業者が関与するレビュー・投稿には「広告」「PR」表記必須   │
│  ├── ギフティング（商品無償提供）によるレビューも対象           │
│  ├── アフィリエイト広告も事業者の責任                           │
│  ├── インフルエンサーへの依頼は全て対象                         │
│  └── 違反した場合の責任は事業者（インフルエンサーではない）     │
│                                                                 │
│  【違反時のリスク】                                             │
│  ├── 措置命令（消費者庁サイトに社名入りで公開される）           │
│  ├── 課徴金（売上の3%）                                        │
│  └── ブランド信頼の不可逆的毀損                                 │
│                                                                 │
│  【二重価格表示の適法条件】                                     │
│  ├── 比較対照価格で過去8週間のうち4週間以上販売実績がある       │
│  ├── セール開始時点から2週間前までに販売実績がある               │
│  └── 最終販売日から2週間以内である                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 薬機法（医薬品医療機器等法）

```
┌─────────────────────────────────────────────────────────────────┐
│              薬機法チェックリスト                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  【基本原則】                                                   │
│  医薬品でないものに、医薬品的な効能効果を謳ってはならない       │
│                                                                 │
│  【化粧品で言えないこと（アウト表現）】                         │
│  ├── 「シミが消える」「シワがなくなる」                         │
│  ├── 「アンチエイジング効果」（具体的効能を示唆する場合）       │
│  ├── 「医療レベルの効果」                                       │
│  └── 「〇〇成分が肌の奥深くに浸透して改善」                    │
│                                                                 │
│  【化粧品で言えること（OK表現）】                               │
│  ├── 「メラニンの生成を抑え、しみ・そばかすを防ぐ」            │
│  ├── 「肌にうるおいを与える」「肌を整える」                     │
│  └── 化粧品の効能効果として厚労省が定める56項目の範囲内        │
│                                                                 │
│  【健康食品・サプリメントで言えないこと（アウト表現）】         │
│  ├── 「飲むだけで痩せる」「血圧が下がる」                       │
│  ├── 「〇〇が治る」「〇〇を予防する」                          │
│  ├── 「全員が効果を実感」                                       │
│  ├── 「医師も推薦」（推薦の文脈によりアウト）                   │
│  └── ビフォーアフター写真で治療効果を暗示                       │
│                                                                 │
│  【美顔器・美容機器の注意点】                                   │
│  ├── 「肌のたるみを改善」「シワを消す」等の効能表現はアウト    │
│  ├── 医療機器に該当する場合、製造販売業許可が必要               │
│  ├── PSEマーク（電気用品安全法）の適合確認                      │
│  └── 技適マーク（電波法）の確認（電波を発する機器の場合）       │
│                                                                 │
│  【LP制作時の注意】                                             │
│  ├── 外注ライター・デザイナーは薬機法を知らない前提でチェック   │
│  ├── 「売れるコピー」は法律のラインを超えやすい                 │
│  └── 公開前に必ず薬機法チェックを実施すること                   │
│                                                                 │
│  【違反時のリスク】                                             │
│  ├── 刑事罰あり（懲役・罰金）                                  │
│  ├── 逮捕・起訴の可能性                                        │
│  └── 景表法と異なり個人の刑事責任を問われる                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```
│              PL法チェックリスト                                    │

│                                                                 │
│  【基本原則】                                                   │

│  ├── サプリメントの健康被害 → 販売元が訴えられる               │
│  └── 食品の異物混入 → 自社ブランドEC事業者が回収費用全額負担   │
│                                                                 │
│  【必須対策】                                                   │
│  □ PL保険（生産物賠償責任保険）に加入済みか                     │
│    → 年間数万円で数千万円の賠償リスクをカバー                   │
│  □ OEM契約に品質保証条項・瑕疵担保条項があるか                  │
│  □ 製造ロットの追跡可能性（トレーサビリティ）を確保しているか   │
│  □ 製品テスト・品質検査の記録を保管しているか                   │
│  □ リコール手順書を作成しているか                               │
│                                                                 │

```

```
│  ├── 自社サイト内でのみ使えるポイント → 自家型前払式支払手段    │
│  ├── 他社サイトでも使えるポイント → 第三者型前払式支払手段      │

│  【違反時のリスク】                                             │
│  ├── 1年以下の懲役 or 300万円以下の罰金                        │
│  ├── 業務改善命令                                               │
│  └── キャッシュフローへの直接影響（供託金）                     │
│                                                                 │
│  【設計段階での対策】                                           │
│  ├── ポイント有効期限を6ヶ月以内に設定（適用除外を狙う）       │
│  ├── 値引きクーポン方式に変更（前払式支払手段に非該当）         │
│  └── ポイント残高を定期的にモニタリングする仕組みを構築         │
│                                                                 │
```


### EC事業者向け法務セルフチェック実行プロトコル

```
【実行タイミング】
├── 新規ECサイト立ち上げ時（必須）
├── 新商品追加時
├── LP・広告クリエイティブ制作時
├── 定期購入サービス開始時
├── ポイント制度導入・変更時
├── 四半期ごとの定期チェック（推奨）
└── 法改正時（CLOが自動監視・アラート）

【チェック優先順位】
1. 特商法（表示不備は即行政処分）
2. 景表法（ステマ規制含む。日常的に違反しやすい）
3. 薬機法（化粧品・健食・美容機器を扱う場合。刑事罰あり）
4. 食品表示法（食品を扱う場合。アレルゲン漏れは即回収）
5. PL法（OEM商品を扱う場合。PL保険加入が最優先）
6. 資金決済法（ポイント制度がある場合）
7. JAS法（加工食品・OEM食品を扱う場合）
```
