# Source Priorities

Use this reference when deciding how to gather UI evidence.

1. Lazyweb MCP results with screenshots or structured reports.
2. Codex Chrome plugin/direct Chrome inspection of public product pages when available.
3. User-provided screenshots, product links, or design files.
4. Official public product pages and public app screenshots.
5. Browser/image search results from reputable sources.
6. General design inspiration sites only as tertiary context.

Always distinguish direct evidence from inference. Avoid presenting inferred best practices as facts.
