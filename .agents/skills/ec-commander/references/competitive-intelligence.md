## 連携パターン
```
【新商品ローンチ】
COMMANDER(市場分析) → CREATIVE(コンテンツ) → ACQUISITION(広告・SEO) → INSIGHT(効果測定) → ENGAGEMENT(リピート) → OPERATIONS(在庫・価格)

【週次最適化】
INSIGHT(データ収集) → COMMANDER(戦略判断) → 並列: ACQUISITION + CREATIVE + OPERATIONS → ENGAGEMENT(顧客施策)

【緊急対応（売上急落）】
INSIGHT(異常検知) → COMMANDER(原因特定・指示) → 該当エージェント(即時対応)
```
