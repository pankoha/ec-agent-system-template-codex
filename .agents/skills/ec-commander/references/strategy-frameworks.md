# 戦略分析フレームワーク詳細

## マーケティングトレース 6ステップ分析

```
【Step 1】企業概要を整理
└── 企業名 / 主要商品・ブランド / 業界カテゴリー / ビジョン・ミッション / 財務状況

【Step 2】市場と競合を定義
├── PEST（外部環境分析）: 政治/経済/社会/技術の影響要因
├── 5FORCES（業界構造分析）: 競争環境・参入障壁・代替品の脅威
└── 3C（市場機会分析）: Customer/Competitor/Company

【Step 3】戦略整理
├── STP: Segmentation/Targeting/Positioning
└── 4P/4C:
    ├── Product / Customer Value
    ├── Price / Cost
    ├── Place / Convenience
    └── Promotion / Communication

【Step 4】KGI/KPIを整理
├── 収益ドライバーの特定: 売上 = 数量 × 単価 × 頻度
└── KGI/KPIツリーの構築: 最終目標から逆算した指標設計

【Step 5】組織資源や構造を整理
├── 組織図（意思決定構造）
├── 組織の強み/弱み
└── 組織資源（人材・資金・ノウハウ）

【Step 6】もし自分がCMOだったら？
├── ターゲット再定義: 現状のターゲットは最適か？
├── ポジショニング再定義: 競合との差別化は明確か？
└── マーケティングミックス再定義: 4Pの最適化余地はあるか？
```

## フェーズ別アプローチ
```
Phase 0（0→3000万）:
├── 目標: PMF達成、月商250万安定
├── 配分: 1チャネル集中（Amazon or 楽天）
├── KPI: CVR、リピート率、レビュー数

Phase 1（3000万→1億）:
├── 目標: 月商830万、勝ちパターン確立
├── 配分: メインチャネル70%、サブ30%
├── KPI: ROAS、CPA、月次成長率

Phase 2（1億→3億）:
├── 目標: 月商2500万、チャネル拡大
├── 配分: 3チャネル並行（Amazon/楽天/自社EC）
├── KPI: チャネル別利益率、新規獲得数

Phase 3（3億→8億）:
├── 目標: 月商6700万、効率化
├── 配分: 4チャネル+B2B開始
├── KPI: LTV、CAC、営業利益率

Phase 4（8億→13億）:
├── 目標: 月商1.08億、利益最大化
├── 配分: 全チャネル最適化+新規事業
├── KPI: ROIC、キャッシュフロー、市場シェア
```

## 目標分解テンプレート
```markdown
## 目標分解: [目標]達成プラン

### フェーズ判定
- 現在: Phase [X]（[売上規模]）
- 目標: Phase [Y]（[目標規模]）

### 全体戦略
- 重点施策: [3つ]
- 必要投資: ¥[金額]

### マイルストーン
| Month | 目標売上 | 重点施策 | 投資額 |

### チャネル別計画
| チャネル | 現状 | 目標 | 増分 | 優先施策 |

### リソース配分
| リソース | 配分 | 用途 |
```

## 予算配分ロジック（フェーズ別）
```
Phase 0: 広告50% / コンテンツ30% / ツール15% / その他5%
Phase 1-2: 広告60% / コンテンツ20% / CRM10% / ツール10%
Phase 3-4: 広告40% / CRM25% / コンテンツ15% / 新規事業10% / ツール10%
```

## フェーズ別KPI
```
Phase 0: 生存指標 → CVR、リピート率、キャッシュフロー
Phase 1: 成長指標 → 月次成長率、ROAS、新規獲得数
Phase 2: 拡大指標 → チャネル別売上、新規/リピート比率
Phase 3: 効率指標 → 営業利益率、LTV/CAC、ROIC
Phase 4: 最適化指標 → 市場シェア、ブランド価値、FCF
```
