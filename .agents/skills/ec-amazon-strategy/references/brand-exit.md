# ブランド売却（M&A）詳細ガイド（Amazon物販3ステップメソッド）

## ブランド売却の概要

```
Amazon自社ブランドの出口戦略:
├── 育てたブランドを第三者に売却して利益を得る
├── 売却価格: 年間営業利益 × 2-4倍が相場
├── 買い手: Amazon FBAアグリゲーター / 個人投資家 / 同業他社
├── ★ 「事業を売却する」という出口があることで、投資対効果が大幅UP
└── ★ 最初から売却を視野に入れて事業を構築する

売却のメリット:
├── まとまったキャッシュの獲得
├── 次の事業への投資原資
├── 事業リスクからの解放
├── ノウハウを活かした新ブランド立ち上げ
└── ★ 「育てる → 売却 → 新ブランド」の繰り返しが可能

売却の条件（買い手が求めるもの）:
├── 安定した月商（最低月商50万円以上、理想100万円+）
├── 健全な利益率（営業利益率20%以上）
├── レビュー資産（100件以上・評価4.0以上）
├── SEO順位の安定（主要KWでTOP 10以内）
├── ブランド登録済み
├── 成長トレンド or 安定した売上推移
└── ★ 「買った後も利益が出続ける」ことの証明が最重要
```

## 売却価値の算定

```
ブランド売却価格の計算:

基本式:
売却価格 = 年間営業利益（SDE） × 倍率

SDE（Seller's Discretionary Earnings）:
├── = 年間売上
│   - 売上原価（商品原価 + FBA手数料 + 販売手数料）
│   - 広告費
│   - ツール費用（Helium10等）
│   - その他変動費
│   + オーナー報酬（オーナーの労働対価を足し戻す）
└── ★ 「新しいオーナーが得られる年間利益」を算出

倍率の目安:
├── 2.0-2.5倍: 小規模ブランド（年商500万未満）
├── 2.5-3.0倍: 中規模ブランド（年商500万-2,000万）
├── 3.0-3.5倍: 大規模ブランド（年商2,000万-5,000万）
├── 3.5-4.0倍+: 優良ブランド（年商5,000万以上 + 成長トレンド）
└── ★ 倍率はブランドの品質・安定性・成長性で変動

倍率を上げる要因:
├── 売上の安定性（月間変動が小さい）
├── 成長トレンド（前年比成長）
├── 高い利益率（30%以上）
├── 多様な商品ライン（単品依存でない）
├── 強いSEO順位（オーガニック比率70%以上）
├── 多くのレビュー（数百件以上）
├── ブランド認知度（指名検索がある）
└── 低いオーナー依存度（自動化されている）

倍率を下げる要因:
├── 売上の季節変動が大きい
├── 単品商品に依存
├── 広告依存度が高い（オーガニック比率低い）
├── レビューが少ない / 評価が低い
├── オーナーの労働時間が多い
└── 競合の脅威が大きい

計算例:
├── 年間売上: ¥12,000,000（月商100万円）
├── 年間SDE: ¥3,600,000（利益率30%）
├── 倍率: 3.0倍
├── 売却価格: ¥10,800,000
└── ★ 月商100万円のブランドが約1,000万円で売却可能
```

## 売却準備

```
売却に向けた準備（6ヶ月前から）:

Step 1: P&L（損益計算書）の整理
├── 月次P&Lを直近12ヶ月分作成
├── すべての収入・経費を正確に記録
├── 個人支出と事業支出を明確に分離
├── 広告費・ツール費用を漏れなく記載
└── ★ 買い手のデューデリジェンスに耐えられる精度

Step 2: ブランド資産の棚卸し
├── 商標登録証
├── Amazon ブランドレジストリの状態
├── 商品ラインナップと在庫状況
├── サプライヤーリストと契約状況
├── 広告アカウントの実績データ
├── 商品画像・A+コンテンツ等のクリエイティブ資産
└── ★ すべての資産を一覧化しておく

Step 3: 事業の最適化（売却前の磨き上げ）
├── 利益率の改善（不要コストの削減）
├── 在庫の最適化（過剰在庫の消化）
├── カタログの最終改善
├── レビュー数の積み増し
├── 広告効率の改善（TACoS低減）
└── ★ 直近3ヶ月のパフォーマンスが売却価格に最も影響

Step 4: オペレーションのドキュメント化
├── 日次・週次・月次のルーティンを文書化
├── 広告運用の手順書
├── 在庫管理・発注の手順書
├── サプライヤーとのコミュニケーション方法
├── LINE/メルマガの運用方法
└── ★ 「引き継ぎが容易」= 倍率UP要因

Step 5: 依存関係の解消
├── オーナー個人への依存を減らす
│   ├── 顧客対応の自動化
│   ├── 広告のルーティン化
│   └── 発注の半自動化
├── 特定サプライヤーへの依存リスク確認
└── ★ 「オーナーが抜けても事業が回る」状態を作る
```

## 売却プラットフォーム

```
Amazon FBAブランドの売却先:

1. M&Aマッチングプラットフォーム
├── ラッコM&A: 国内最大級のサイトM&Aプラットフォーム
│   ├── 手数料: 売り手無料 / 買い手5%
│   └── FBAブランドの取引実績あり
├── バイトM&A: EC事業特化
├── Flippa: 海外プラットフォーム（英語）
├── Empire Flippers: 海外（高額案件に強い）
└── ★ まず国内プラットフォームから検討

2. FBAアグリゲーター
├── 概要: 複数のAmazonブランドを買収する企業
├── 海外: Thrasio / Perch / Heyday（ただし市場縮小傾向）
├── 国内: 数社が活動中
├── メリット: 買収の意思決定が速い / 事業理解がある
└── ★ 直接アプローチ or M&A仲介経由

3. M&A仲介会社
├── 手数料: 売却額の5-15%
├── メリット: 買い手探し〜契約まで一括サポート
├── デメリット: 手数料が高い
└── ★ 売却額が1,000万円以上の場合に検討

4. 直接売却
├── 同業者・知人への売却
├── SNS/コミュニティでの募集
├── メリット: 手数料ゼロ
├── デメリット: 買い手探しの手間 / 契約リスク
└── ★ 信頼できる相手がいる場合のみ推奨
```

## デューデリジェンス対応

```
買い手が確認する項目（デューデリジェンス）:

財務面:
├── 直近12-24ヶ月のP&L
├── Amazon Seller Central のレポート（売上・費用の裏付け）
├── 広告費の実績レポート
├── 在庫金額・回転率
├── キャッシュフロー
└── ★ 数字の正確性が最重要。嘘は必ずバレる

事業面:
├── 商品ラインナップ / 各商品のパフォーマンス
├── SEOランキング / オーガニック比率
├── レビュー数・評価
├── 広告のACoS / TACoS推移
├── サプライヤー情報 / MOQ / リードタイム
├── ブランド登録状況 / 商標
└── 成長トレンド / 季節性

リスク面:
├── Amazon規約違反の履歴
├── アカウントヘルスの状態
├── 知的財産リスク
├── サプライヤーの安定性
├── 競合の脅威レベル
└── 在庫の品質・経年劣化リスク

提出を求められるデータ:
├── セラーセントラルへのゲストアクセス権
├── 広告コンソールのデータエクスポート
├── P&Lスプレッドシート
├── サプライヤーとの契約書/発注書
├── 商標登録証
└── ★ 事前に準備しておくとスムーズ
```

## 売却プロセス

```
売却の流れ:

Step 1: 売却準備（1-3ヶ月）
├── P&L整理・資産棚卸し
├── 事業の磨き上げ
├── 売却価格の希望額設定
└── 売却プラットフォーム/仲介の選定

Step 2: 買い手探し（1-2ヶ月）
├── プラットフォームへの掲載 or 仲介に依頼
├── 秘密保持契約（NDA）の締結
├── 事業概要の共有
└── 関心のある買い手との初期面談

Step 3: デューデリジェンス（2-4週間）
├── 買い手による詳細調査
├── データ・レポートの共有
├── Q&A対応
└── 価格交渉

Step 4: 契約・クロージング（2-4週間）
├── 売買契約書の作成
├── 支払い条件の合意
├── エスクローサービスの利用（推奨）
├── アカウント・資産の移転準備
└── 引き継ぎ計画の合意

Step 5: 引き継ぎ・移転（2-4週間）
├── セラーセントラルアカウントの移転
├── ブランドレジストリの移転
├── サプライヤーへの紹介・引き継ぎ
├── 運用マニュアルの引き渡し
├── 広告アカウントの引き継ぎ
└── サポート期間（通常30-90日）

★ 全体で3-6ヶ月が一般的なタイムライン
★ エスクローサービスの利用で代金未払いリスクを回避
```

## 売却を見据えた事業構築

```
最初から売却を見据えて構築すべきポイント:

1. 記録を残す
├── 月次P&Lを初月から作成
├── 広告データを定期的にエクスポート保存
├── 重要な意思決定の記録
└── ★ 後から遡って作成するのは困難

2. ブランドを作る
├── 商標登録は早期に実施
├── ブランドレジストリに登録
├── ブランドストーリーを構築
├── 統一されたブランドイメージ
└── ★ 「ブランド」が売却価値の中核

3. 自動化する
├── 広告のルーティンを体系化
├── 在庫管理を仕組み化
├── 顧客対応を自動化
└── ★ オーナー依存度が低いほど高く売れる

4. 複数商品を持つ
├── 単品依存はリスクが高い（買い手目線）
├── 3-5商品のラインナップ
├── 各商品が相互に補完する構成
└── ★ 「ポートフォリオ」としての価値を持たせる

5. オーガニック比率を上げる
├── 広告依存の事業は評価が低い
├── オーガニック売上70%以上を目指す
├── SEO順位の安定性を示す
└── ★ 「広告を止めても売れ続ける」が最大の評価ポイント
```
