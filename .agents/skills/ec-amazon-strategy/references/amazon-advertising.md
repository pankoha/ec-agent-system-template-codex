# Amazon広告運用 詳細ガイド（Amazon物販3ステップメソッド）

## 広告の本質: 「SEOのための手段」

```
Amazon広告の正しい位置づけ:

❌ 間違い: 「広告で利益を出す」
✅ 正解: 「広告でSEOスコアを上げ、オーガニック売上で利益を出す」

広告は「カタログを育てるサプリメント」:
├── 広告経由の注文 → 実績SEOスコアに加算
├── SEOスコア向上 → オーガニック検索順位UP
├── オーガニック順位UP → 広告なしでも売れる
├── 最終目標: オーガニック売上比率70%以上
└── 広告は目的ではなく手段

★ 広告単体の収益性（ACoS）だけで判断してはならない
★ 「オーガニック込み」の総合収益性で評価する
```

## 広告の種類と使い分け

### SP広告（スポンサープロダクト広告）
```
SP広告 = メインの広告手段:
├── 表示場所: 検索結果ページ・商品詳細ページ
├── 課金方式: CPC（クリック課金）
├── 目的: 特定キーワードでの注文獲得 → SEOスコア構築
│
├── ターゲティングの種類:
│   ├── オートターゲティング（自動）
│   │   ├── Amazon AIが自動でキーワードをマッチング
│   │   ├── 初期データ収集に最適
│   │   ├── 4つのマッチタイプ: ほぼ一致 / 大まかな一致 / 代替商品 / 補完商品
│   │   └── ★ 最初の2週間はオートで開始
│   │
│   └── マニュアルターゲティング（手動）
│       ├── キーワードターゲティング: 特定KWに入札
│       │   ├── 完全一致: そのKWのみ（最高精度）
│       │   ├── フレーズ一致: KW含む検索語句
│       │   └── 部分一致: 関連する検索語句全般
│       ├── 商品ターゲティング: 特定ASIN/カテゴリに表示
│       └── ★ オートのデータを分析してマニュアルに移行
│
└── 運用の流れ:
    ├── Week 1-2: オートターゲティングで開始（日予算¥1,000-3,000）
    ├── Week 3-4: 検索語句レポートを分析
    ├── Week 5+: 効果の高いKWをマニュアルに移行
    └── 継続: 週次で8ステップチューニング実行
```

### SB広告（スポンサーブランド広告）
```
SB広告 = ブランド認知・指名検索獲得:
├── 表示場所: 検索結果ページの上部（最も目立つ位置）
├── 前提条件: ブランド登録（Brand Registry）必須
├── 形式: ブランドロゴ + 見出し + 商品3点
│
├── 活用タイミング:
│   ├── Phase 1後半〜: ブランド認知を高めたい段階
│   ├── 商品ラインが3点以上ある場合に特に有効
│   └── 指名検索（ブランド名検索）を増やしたい場合
│
├── 効果:
│   ├── ブランドストアへの誘導
│   ├── 複数商品の同時訴求
│   ├── 検索結果最上部の独占
│   └── ブランド認知 → 指名検索増 → オーガニックCVR向上
│
└── 注意:
    ├── SP広告より CPC が高い傾向
    ├── 直接的なSEO効果はSP広告の方が大きい
    └── ★ SP広告が安定してからSBを追加
```

### SD広告（スポンサーディスプレイ広告）
```
SD広告 = リターゲティング・競合対策:
├── 表示場所: 商品詳細ページ・Amazon内外のディスプレイ枠
├── 前提条件: ブランド登録必須
│
├── ターゲティング:
│   ├── オーディエンスターゲティング:
│   │   ├── リマーケティング（自社ページ訪問者）
│   │   ├── Amazon オーディエンス（興味関心ベース）
│   │   └── ★ 自社ページ訪問者のリターゲティングが最も効率的
│   │
│   └── 商品ターゲティング:
│       ├── 特定ASINの商品ページに広告表示
│       ├── 競合商品ページへの表示が可能
│       └── ★ 競合から自社への切り替え促進
│
├── 活用タイミング:
│   ├── Phase 2〜: SP/SBが安定した後に追加
│   ├── 競合対策として即効性がある
│   └── リターゲティングでCVR向上
│
└── 注意:
    ├── CTRはSPより低い傾向
    ├── 認知〜検討段階への影響
    └── ★ SP広告と組み合わせて効果を最大化
```

## 8ステップ チューニング手順

```
Amazon広告の週次チューニング（最重要ルーティン）:

┌─────────────────────────────────────────────────────────────┐
│            8ステップ チューニング手順                          │
├─────────────────────────────────────────────────────────────┤
│                                                               │
│  Step 1: 全体数値の確認                                       │
│  ├── 広告費合計 / 売上合計 / ACoS / ROAS                     │
│  ├── 先週比・先月比での変動                                   │
│  └── 予算消化率（予算が足りているか/余っているか）            │
│                                                               │
│  Step 2: キャンペーン別の確認                                 │
│  ├── 各キャンペーンのパフォーマンス比較                       │
│  ├── オート vs マニュアルの効率比較                            │
│  ├── 効率の良いキャンペーン / 悪いキャンペーンの特定           │
│  └── 予算配分の見直し                                         │
│                                                               │
│  Step 3: ターゲティング別の確認                               │
│  ├── キーワード別 / ASIN別のパフォーマンス                    │
│  ├── クリック数50以上のターゲティングを分析対象とする          │
│  ├── 高ACoSターゲティング → 入札下げ or 停止                  │
│  ├── 低ACoSターゲティング → 入札上げ or 予算追加              │
│  └── ★ 最低50クリックないと統計的に判断できない               │
│                                                               │
│  Step 4: 思考（未来を見据えた判断）                           │
│  ├── 「このキーワードは将来オーガニックで上位を取りたいか？」 │
│  ├── YES → 短期的にACoSが高くても投資を継続                   │
│  ├── NO → 効率重視で入札調整                                  │
│  └── ★ 目先のACoSだけで判断しない。SEO投資の視点が重要        │
│                                                               │
│  Step 5: オーガニック込みの数値確認                           │
│  ├── 広告経由売上 + オーガニック売上 の合計で評価              │
│  ├── TACoS（Total ACoS）= 広告費 / 総売上 で判断             │
│  ├── TACoSが下がっている = SEO効果が出ている                  │
│  └── ★ ACoSではなくTACoSで広告の成功を測る                   │
│                                                               │
│  Step 6: 広告順位の確認                                       │
│  ├── 主要KWでの広告掲載順位                                   │
│  ├── TOPページ掲載率                                          │
│  ├── 1ページ目に表示されているか                              │
│  └── 表示されていない場合 → 入札を上げる必要あり              │
│                                                               │
│  Step 7: 自然検索順位の確認                                   │
│  ├── 主要KWでのオーガニック検索順位                            │
│  ├── 先週からの変動                                           │
│  ├── 順位上昇 → 広告効果が出ている証拠                        │
│  ├── 順位横ばい/下落 → 広告投資の増額 or テキストSEO見直し    │
│  └── ★ 広告の最終目的はこの自然検索順位の向上                 │
│                                                               │
│  Step 8: 拡張ターゲティング・検索用語の確認                   │
│  ├── オートターゲティングの検索語句レポート                    │
│  ├── 新しく効果の高いKWを発見 → マニュアルに追加              │
│  ├── 不要なKW → 除外キーワードに追加                          │
│  └── 新しいキャンペーン/広告グループの作成判断                │
│                                                               │
└─────────────────────────────────────────────────────────────┘

チューニングの鉄則:
├── 頻度: 週1回（毎週同じ曜日に実施）
├── 分析最低条件: 50クリック以上のデータ
├── 入札調整幅: 1回あたり±10-20%（急激な変更は避ける）
├── 除外KW追加: ACoS 100%超 かつ 50クリック以上 → 除外候補
└── 新KW追加: CVR 5%以上 かつ 注文3件以上 → マニュアルに追加
```

## 広告収益性ロードマップ

```
広告単体での収益性推移（目安）:

1ヶ月目: 広告収益性 -30% 〜 -50%（赤字投資期）
├── 状態: オートターゲティング中心、データ収集フェーズ
├── ACoS: 80-150%
├── やること: データ収集、無駄なKWの除外開始
└── 心構え: 赤字は想定通り。焦らない

2ヶ月目: 広告収益性 -15% 〜 -20%（改善期）
├── 状態: マニュアルターゲティング追加、効率化開始
├── ACoS: 50-80%
├── やること: マニュアルKW追加、入札最適化、除外KW拡大
└── オーガニック売上が少しずつ増加し始める

3ヶ月目: 広告収益性 -5% 〜 -10%（安定期）
├── 状態: 広告効率化進行、オーガニック売上増加
├── ACoS: 30-50%
├── やること: 入札の精密調整、新KW探索、SB広告検討
└── オーガニック比率50%前後に到達

4ヶ月目: 広告収益性 0% 〜 -5%（損益分岐点）
├── 状態: 広告+オーガニックでほぼブレークイーブン
├── ACoS: 25-35%
├── やること: 利益最大化のための微調整
└── オーガニック比率60-70%

5ヶ月目以降: 広告収益性 黒字化
├── 状態: オーガニック売上がメイン
├── TACoS: 10-15%
├── やること: 広告は維持目的、新KW開拓、SD広告追加検討
└── オーガニック比率70%以上
```

## TACoS（Total ACoS）で評価する

```
ACoS vs TACoS:

ACoS = 広告費 / 広告経由売上 × 100
├── 広告単体の効率を見る指標
├── ACoS 30% = 広告費100円で売上333円
└── ★ この指標だけでは広告の真の効果がわからない

TACoS = 広告費 / 総売上（広告+オーガニック）× 100
├── 広告が事業全体に与える効果を見る指標
├── TACoS 10% = 広告費100円で総売上1,000円
└── ★ こちらの指標で広告の真の効果を判断する

例:
├── 広告費: ¥50,000/月
├── 広告経由売上: ¥150,000 → ACoS 33%
├── オーガニック売上: ¥350,000
├── 総売上: ¥500,000 → TACoS 10%
└── ★ ACoS 33%は高く見えるが、TACoS 10%は優秀

TACoSの推移が重要:
├── TACoS下降 = 広告投資がSEOに効いている（良い傾向）
├── TACoS横ばい = 広告依存から脱却できていない
├── TACoS上昇 = 広告効率が悪化。テコ入れ必要
└── ★ 月次でTACoSの推移を追跡すること
```

## 除外キーワード戦略

```
除外キーワードの運用:

除外すべきキーワード:
├── 無関係なKW: 自社商品と関係のない検索語句
├── 高ACoS KW: ACoS 100%超 かつ 50クリック以上
├── ゼロCVR KW: 50クリック以上で注文ゼロ
├── ブランド名KW: 他社ブランド名での表示（意図しない場合）
└── カテゴリ違いKW: 別カテゴリの商品への表示

除外の手順:
├── Step 1: 検索語句レポートをダウンロード（週次）
├── Step 2: クリック数50以上のKWを抽出
├── Step 3: ACoSでソート → 高ACoSから確認
├── Step 4: 除外判断（不要KWを除外KWリストに追加）
└── Step 5: 除外後の効果を翌週確認

注意:
├── ❌ クリック数が少ない段階で除外しない（判断材料不足）
├── ❌ SEO目標のKWを安易に除外しない（Step 4の思考が重要）
└── ✅ 除外は「完全一致」を基本とする（フレーズ一致は慎重に）
```

## 予算管理

```
広告予算の考え方:

初期予算の目安:
├── 日予算: ¥1,000 〜 ¥5,000（商品単価による）
├── 月予算: ¥30,000 〜 ¥150,000
├── 基準: 想定月商の15-30%を広告費として投資
└── 例: 月商50万円目標 → 広告費¥75,000-150,000/月

予算配分:
├── SP広告: 全体の70-80%（メイン）
├── SB広告: 全体の10-20%（Phase 1後半〜）
├── SD広告: 全体の5-10%（Phase 2〜）
└── ★ 初期はSP広告100%でOK

予算調整のタイミング:
├── 増額: ACoS改善傾向 + 在庫十分 + 予算が早期に消化
├── 減額: オーガニック比率70%以上 + SEO順位安定
├── 一時増額: 大型セール前（在庫積み増しと連動）
└── 緊急減額: 在庫切れリスク時（広告停止も検討）
```
