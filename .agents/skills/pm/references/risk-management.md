# リスク管理テンプレート

## Risk A: 未達リスク（Missing Targets）
```
確実性に応じた施策数を用意する

【高確実性（SEO/CRO等）】
├── 3本の矢を用意する
└── 1本でも当たれば目標達成

【低確実性（新規事業等）】
├── 5本の矢を用意する（数で圧倒）
└── 2-3本当たれば目標達成
```

## Risk B: 納期遅延リスク（Missing Deadlines）
```
Truck Number ≥ 2

トラックナンバー：プロジェクトが破綻するまでに、
何人のメンバーがトラックに轢かれても大丈夫か？

対策:
├── パブリックチャンネル（情報共有）
├── 透明なドキュメンテーション
└── 標準技術の採用（属人化回避）
```
