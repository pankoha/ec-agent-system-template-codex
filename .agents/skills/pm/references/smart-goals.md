# SMART目標設定 & ROI計算テンプレート

## SMARTの法則
```
S - Specific（具体的）
M - Measurable（計測可能）
A - Achievable（達成可能）
R - Realistic（現実的）
T - Time-bound（期限付き）
```

## 悪い例 vs 良い例
```
❌ 「売上を上げる」
✅ 「2025年12月までに店舗売上を月1000万円純増させる」

❌ 「SNSを頑張る」
✅ 「2025年6月までにInstagramフォロワー10,000人、エンゲージメント率3%以上」
```

## KPI設計の複眼思考
```
KPI = Quantity（数） × Rate（率）

例: 売上 = アクセス数 × CVR × 客単価
├── 数を増やす施策: 広告、SEO、SNS
└── 率を上げる施策: LP改善、UX改善、価格最適化

両方を同時に追わない。フェーズで使い分ける。
```

## ROI計算

### 成功基準: ROI 300%
```
ROI = (成果 - 投資) / 投資 × 100

100% = コンサルタントフィー（Break-even）
200% = クライアント内部コスト（Labor/Time）
300% = 真の価値創出（Success Zone）
```

### ROI計算シート テンプレート
```markdown
### 投資（Input）
| 項目 | 金額 |
|------|------|
| コンサルティングフィー | ¥ |
| クライアント工数（時給×時間） | ¥ |
| ツール・広告費 | ¥ |
| **合計投資** | ¥ |

### 成果（Output）
| 項目 | 金額 |
|------|------|
| 売上増加 | ¥ |
| コスト削減 | ¥ |
| その他価値 | ¥ |
| **合計成果** | ¥ |

### ROI計算
ROI = (成果 - 投資) / 投資 × 100 = **[X]%**

### 判定
- [ ] 300%以上: Success Zone
- [ ] 200-300%: Acceptable
- [ ] 100-200%: Needs Improvement
- [ ] 100%以下: Failure（あなたは負債）
```

## 品質管理 QCD
```
Quality = Output / (Cost × Comparison Subject)

Q (Quality): スコープと深さ - 何をどこまでやるか
C (Cost): フィー - 予算内での最大価値提供
D (Delivery): 頻度とスピード - 納品タイミング

品質チェックリスト:
□ スコープは明確に定義されているか
□ 品質基準は合意されているか
□ 比較対象（期待値）を把握しているか
□ 納品物は基準を満たしているか
□ コストは予算内に収まっているか
□ 納期は守られているか
```
