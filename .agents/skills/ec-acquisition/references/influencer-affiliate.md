# インフルエンサー & アフィリエイト管理

## インフルエンサースコアリング基準（100点満点）
```
├── リーチ（20点）: フォロワー数
├── エンゲージメント（30点）: いいね/コメント率
├── 適合性（25点）: ターゲット一致度
├── 信頼性（15点）: 過去実績、評判
└── コスト効率（10点）: CPE、予算内
```

## ティア分類
```
├── Nano（1K-10K）: 高エンゲージ、低コスト
├── Micro（10K-50K）: バランス良好
├── Mid（50K-500K）: リーチ拡大
├── Macro（500K-1M）: 認知向上
└── Mega（1M+）: ブランディング
```

## インフルエンサーROI計算
```
ROI = (売上 - 報酬 - 商品コスト) / 総コスト × 100
目標: 200%以上
```

## アフィリエイトコミッション設計

### 商品別コミッション
```
├── 高利益商品: 8-10%
├── 標準商品: 5-7%
├── 低利益商品: 3-5%
└── 新商品: 10-15%（期間限定）
```

### パフォーマンス別
```
├── トップパートナー: 基本+2%
├── 成長パートナー: 基本+1%
└── 新規パートナー: 基本
```

### 成果指標
```
├── EPC（クリック単価）
├── CVR（コンバージョン率）
├── AOV（平均注文額）
└── キャンセル率
```
