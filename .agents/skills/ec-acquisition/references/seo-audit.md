# テクニカルSEO監査チェックリスト

## Core Web Vitals
```
□ LCP < 2.5秒（最大コンテンツ描画）
□ FID < 100ms（初回入力遅延）
□ CLS < 0.1（累積レイアウトシフト）
□ INP < 200ms（次回ペイント応答）
```

## クロール・インデックス
```
□ robots.txt最適化
□ XMLサイトマップ送信
□ noindex/nofollow適切設定
□ 重複コンテンツ対策（canonical）
□ 404エラー解消
```

## 構造化データ
```
□ Product（商品）
□ BreadcrumbList（パンくず）
□ FAQPage（FAQ）
□ Review（レビュー）
□ Organization（組織）
```

## モバイル対応
```
□ レスポンシブデザイン
□ タップターゲットサイズ
□ フォントサイズ
□ ビューポート設定
```

## キーワード分類
```
購入意図別:
├── Transactional（今すぐ客）: 「〇〇 通販」「〇〇 購入」→ 商品ページ、広告
├── Commercial（検討客）: 「〇〇 おすすめ」「〇〇 比較」→ 比較記事、ランキング
├── Informational（情報収集）: 「〇〇 効果」「〇〇とは」→ ブログ記事、ハウツー
└── Navigational（指名検索）: 「ブランド名」→ ブランドページ

クラスタリング:
├── メインKW（検索Vol高、競合強）
├── サブKW（検索Vol中、競合中）
├── ロングテール（検索Vol低、競合弱）
└── 関連KW（共起語、LSI）
```
