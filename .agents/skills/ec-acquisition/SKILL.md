- 勝者基準: CTR +10% / CVR +5% / CPA -10% / ROAS +15%

### Skill 6: Influencer Scoring & Matching（インフルエンサー評価）

## 解像度プロトコル — 集客の解像度（ACQUISITION必須）

### 集客課題のWhy So?ループ
```
集客施策を策定する前に、課題の根本原因を特定するWhy So?ループを実施すること。

例）「CPAが高騰している」
├── Why So? → クリック単価が上昇している
├── Why So? → オークション競争が激化している
├── Why So? → 新規参入の競合が同じキーワードに出稿開始
├── Why So? → 市場が成熟期に入り、差別化が困難に
└── Why So? → 自社の訴求が競合と同質化している
→ 根本原因: 訴求の差別化不足（「入札を上げる」では解決しない）

❌ 禁止: 「CPAが高い → 予算を増やそう / 入札を下げよう」（1段階で結論）
✅ 必須: 最低5回のWhy So?で根本原因を特定してから打ち手を決定
```

### 競合比較の解像度チェック
```
競合分析時に以下の症状が出たら、解像度が低いと判断すること:

⚠️ 「競合に全部勝っている」
→ 比較軸の設定が甘い。顧客視点の価値軸で再比較すること
→ 顧客は何を基準に選んでいるか？価格/品質/ブランド/利便性の重み付けは？

⚠️ 「競合はいません」
→ 顧客は別の手段で課題を解決している。既存代替手段を調査すること
→ 「何もしない」「別の方法で解決する」も最大の競合

⚠️ 「広告を出せば売れる」
→ なぜ今まで出さなかったのか？構造的障壁を分析すること
→ 広告で獲得した顧客のLTVは十分か？単発利益ではなく構造的に検証

⚠️ 「どのチャネルでも通用する」
→ チャネル特性の理解が不足。各チャネルのユーザー心理を個別に分析
```

## Review & Challenge（壁打ち・批判的検証）

集客施策の品質を担保するため、Sub-Agentの提案を多角的に検証する。

### ACQUISITION固有の検証レンズ
```
① チャネル適切性: そのチャネルは本当にターゲットに届くか？根拠は？
② ROAS/CPA妥当性: 目標ROAS達成の根拠は？類似事例・過去データはあるか？
③ クリエイティブ疲弊: 同じ訴求を使い続けていないか？テスト頻度は十分か？
④ 競合差別化: 競合と同じ手法に陥っていないか？差別化要素は何か？
⑤ スケーラビリティ: この施策は予算2倍にしても効率を維持できるか？
⑥ 解像度チェック: 課題のWhy So?は十分か？表面的な指標悪化の裏にある構造的原因を特定しているか？
```

### Sub-Agentへのフィードバック例
```
❌ 「このキーワードでいきましょう」（検証なき同意は禁止）
✅ 「CPC ¥80の想定だが、直近3ヶ月の実績CPCは？競合の入札状況も確認して」
✅ 「Meta広告のLAL 1%だけでなく、3%と5%の比較テスト設計も含めて再提案して」
✅ 「インフルエンサーA案のROI試算に商品コストが含まれていない。全コスト込みで再計算」
✅ 「CPA高騰の対策が入札調整だけ。Why So?で根本原因を特定し、訴求差別化も検討して」
```
