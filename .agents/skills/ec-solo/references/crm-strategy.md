# CRM・顧客関係管理 詳細ガイド

## 「友人関係」アプローチ

### 顧客との距離感
```
核心思想: 顧客を「お客様」ではなく「友人」として扱う

友人関係アプローチの原則:
├── 1. フォーマルすぎない文体
│   ├── ❌ 「拝啓 平素より大変お世話になっております」
│   ├── ✅ 「○○です！今日はいいお知らせがあります」
│   └── トーン: 丁寧だが親しみやすい
│
├── 2. 頻繁に連絡する
│   ├── 友人なら毎日LINEしても嫌がらない
│   ├── 距離が空くと疎遠になる
│   └── ★ 配信頻度 = 関係性の深さ
│
├── 3. 役に立つ情報を届ける
│   ├── セール情報だけでなく、使い方のコツ
│   ├── 業界の豆知識
│   └── 季節のおすすめ
│
├── 4. 返信しやすい雰囲気を作る
│   ├── 「感想を教えてください！」
│   ├── 「質問があればいつでもどうぞ」
│   └── 返信が来たら必ず返す
│
└── 5. 失敗や裏話も共有する
    ├── 「実は開発中にこんな失敗が…」
    ├── 完璧なブランドイメージより人間味
    └── 共感 → 信頼 → リピート
```

## 頻度重視の配信戦略

### 200文字×5回 > 1,000文字×1回
```
短文配信の威力:

200文字メールの実例:

【例1: 使い方のコツ】
「○○です！
実は△△を○○すると効果が2倍になるって知ってました？
お客様から教えていただいたテクニックなんですが、
めちゃくちゃ良かったのでシェアします。
詳しくはこちら→ [リンク]」
（120文字）

【例2: お客様の声】
「○○です！
今朝、お客様からこんなメッセージをいただきました。
『△△を使い始めてから□□が改善しました！』
すごく嬉しかったのでシェアします。
○○さんもぜひ試してみてください！」
（100文字）

【例3: 裏話】
「○○です！
実は新商品の開発中に大失敗しまして…（笑）
△△が□□になってしまったんです。
でもそのおかげで○○という発見がありました。
次のメールで詳しくお話ししますね！」
（110文字）

メリット:
├── 書くのに5分しかかからない
├── 読むのに30秒で済む（開封率UP）
├── 毎日配信しても負担が小さい
└── 「次も読もう」と思わせる気軽さ
```

## カゴ落ち対策

### カゴ落ちメール最適化
```
カゴ落ちメール設定（必須）:

タイミング:
├── 1通目: カゴ落ち1時間後 ★最重要
│   ├── 件名: 「お忘れ物はありませんか？」
│   ├── 内容: 商品画像 + 「カートに商品が残っています」
│   ├── CTA: 「お買い物を続ける」→ カートページ直リンク
│   └── 効果: カゴ落ち回収の60-70%はこの1通目
│
├── 2通目: カゴ落ち24時間後
│   ├── 件名: 「まだ間に合います！○○が気になっていませんか？」
│   ├── 内容: 商品のベネフィット再訴求 + レビュー紹介
│   ├── CTA: 「今すぐ購入する」
│   └── オプション: 送料無料クーポン or 5%OFFクーポン
│
└── 3通目: カゴ落ち72時間後（オプション）
    ├── 件名: 「最後のご案内です」
    ├── 内容: 在庫残りわずか or 限定クーポン
    ├── CTA: 「クーポンを使って購入する」
    └── ★ 3通目以降は効果が薄いので3通まで

設定方法:
├── Shopify標準: 設定 → チェックアウト → カゴ落ちメール（1通のみ）
├── Omnisend: ワークフロー → カゴ落ち → 3通シリーズ設定
└── Klaviyo: フロー → Abandoned Cart → 条件分岐付き設定
```

### カゴ落ちの原因と対策
```
カゴ落ち原因（多い順）:
├── 1位: 送料が想定外に高い（48%）
│   └── 対策: 送料を商品ページで事前表示 / 送料無料ライン設定
│
├── 2位: 会員登録が必要だった（24%）
│   └── 対策: ゲストチェックアウト有効化
│
├── 3位: チェックアウトが複雑（18%）
│   └── 対策: 入力項目の削減 / 住所自動入力
│
├── 4位: 決済方法がなかった（13%）
│   └── 対策: Komoju導入（PayPay/コンビニ/銀行振込）
│
├── 5位: 納期が不明or遅い（12%）
│   └── 対策: 「○日以内にお届け」の明示
│
├── 6位: 合計金額が高すぎた（10%）
│   └── 対策: セット割引 / 初回クーポン
│
└── 7位: 単純に忘れた（8%）
    └── 対策: カゴ落ちメール（1時間後）
```

## LINE連携

### LINE活用の基本設計
```
LINEの役割（EC特化）:
├── 1. 問い合わせ対応
│   ├── チャットでの質問対応
│   ├── 購入前の不安解消
│   └── 注文後のフォロー
│
├── 2. リマインド配信
│   ├── カゴ落ちリマインド
│   ├── 再入荷通知
│   ├── セール開始通知
│   └── 定期便のリマインド
│
├── 3. セグメント配信
│   ├── 購入者限定メッセージ
│   ├── 未購入者向けオファー
│   └── VIP顧客向け先行案内
│
└── 4. 1to1コミュニケーション
    ├── 誕生日メッセージ
    ├── 購入お礼メッセージ
    └── 使い方フォロー

LINE友だち獲得の導線:
├── サンクスページ: 「LINEで使い方のコツをお届け」
├── 同梱チラシ: QRコード + 友だち追加特典
├── メルマガ: 「LINEでも情報配信中」
├── SNSプロフィール: 友だち追加リンク
└── Meta広告: LP → LINE追加の導線
```

### LINE配信のベストプラクティス
```
配信頻度と内容:
├── 推奨頻度: 週2-3回
├── 配信時間: 12:00-13:00 or 19:00-21:00
│
├── コンテンツミックス:
│   ├── 40%: 商品情報・セール告知
│   ├── 30%: 使い方のコツ・お役立ち情報
│   ├── 20%: お客様の声・レビュー紹介
│   └── 10%: 裏話・開発ストーリー
│
└── リッチメッセージの活用:
    ├── 画像 + テキスト + CTAボタン
    ├── カルーセル形式（複数商品紹介）
    └── ★ テキストのみよりCTR 2-3倍
```

## サンクスページ活用

### 購入後の黄金タイミング
```
サンクスページ（購入完了ページ）での施策:

最優先（必須設定）:
├── 1. 自己紹介動画（30-60秒）
│   ├── 「ご購入ありがとうございます！」
│   ├── 「○○と申します。開発者です」
│   ├── 「使い方のコツは…」
│   └── → 顔を見せることで信頼構築
│
├── 2. LINE友だち追加
│   ├── 「使い方のコツをLINEでお届けします」
│   ├── QRコード + ワンクリック追加ボタン
│   └── ★ 購入直後の追加率は最も高い
│
└── 3. SNSフォロー
    ├── Instagram / X / Facebook
    └── 「最新情報はSNSで配信中」

推奨（効果的）:
├── 4. アンケート（「どこで知りましたか？」）
│   └── マーケティングチャネルの効果測定
├── 5. 次回購入クーポン
│   └── 「次回使える10%OFF（30日以内有効）」
└── 6. おすすめ商品表示
    └── 関連商品のクロスセル

ReConvertアプリで全て実装可能
```

## セグメント配信

### 基本セグメント設計
```
必須セグメント（最低限）:
├── セグメント1: 購入者
│   ├── 配信内容: 使い方のコツ、レビュー依頼、リピート促進
│   ├── トーン: 感謝 + 関係深化
│   └── 頻度: 週3-5回
│
├── セグメント2: 未購入者（メルマガ登録のみ）
│   ├── 配信内容: 商品紹介、お客様の声、初回オファー
│   ├── トーン: 興味喚起 + 不安解消
│   └── 頻度: 週3回
│
├── セグメント3: カゴ落ちユーザー
│   ├── 配信内容: カゴ落ちメール + 限定オファー
│   ├── トーン: リマインド + インセンティブ
│   └── 頻度: カゴ落ち後3通（自動配信）
│
└── セグメント4: VIP（2回以上購入者）
    ├── 配信内容: 先行案内、限定セール、新商品モニター依頼
    ├── トーン: 特別感 + パートナーシップ
    └── 頻度: 週3-5回 + 限定配信

発展セグメント（Omnisend / Klaviyo移行後）:
├── 購入金額帯別（高額/中額/低額）
├── 購入頻度別（月1回/3ヶ月に1回/半年に1回）
├── 商品カテゴリ別（カテゴリAを買った人/Bを買った人）
├── 離脱リスク別（最終購入から○日経過）
└── 流入チャネル別（広告経由/オーガニック/SNS経由）
```

## リピート施策

### リピート率向上の仕組み
```
リピート促進のフレームワーク:

Phase 1での目標: リピート率15%
Phase 3での目標: リピート率30%以上

リピート促進施策の優先順位:
├── 1. 購入後フォローメールシリーズ（自動配信）
│   ├── 購入翌日: お礼 + 使い方ガイド
│   ├── 購入3日後: 「届きましたか？」確認
│   ├── 購入7日後: 使い方のコツ
│   ├── 購入14日後: レビュー依頼
│   └── 購入30日後: リピート購入オファー
│
├── 2. 定期便/サブスクリプション
│   ├── 消耗品の場合: 定期便割引10-15%
│   ├── 初回50%OFF → 2回目以降10%OFF
│   └── 解約率を下げるためのコミュニケーション
│
├── 3. ポイント/リワードプログラム
│   ├── Shopifyアプリ: Smile.io等
│   ├── 購入金額の1-5%をポイント還元
│   └── ★ Phase 2以降で検討（初期は不要）
│
├── 4. 限定・先行案内
│   ├── 新商品の先行販売
│   ├── リピーター限定セール
│   └── 「あなただけに」の特別感
│
└── 5. 同梱物の工夫
    ├── 手書き風のお礼カード
    ├── 次回使えるクーポン
    ├── 使い方ガイドの小冊子
    ├── サンプル品の同梱
    └── LINE/SNSフォローのQRコード
```

## 問い合わせ対応

### 問い合わせを購入機会にする
```
問い合わせ対応のマインドセット:
├── 問い合わせ = 購入意欲の表れ
├── 迅速な対応 = CVR向上
└── 丁寧な対応 = LTV向上

対応チャネルの優先度:
├── 1. LINE（最速・最簡単）
│   └── 目標: 30分以内に返信
├── 2. メール
│   └── 目標: 24時間以内に返信
├── 3. チャットボット（Chatty）
│   └── FAQ自動応答 → 有人対応へ切替
└── 4. 電話（必要に応じて）

よくある問い合わせと対応テンプレート:
├── 「サイズが合うか不安」→ サイズガイド + 返品保証の案内
├── 「届くまでどのくらい？」→ 具体的な日数 + 追跡番号の説明
├── 「他の商品との違いは？」→ 比較表 + おすすめの提案
├── 「プレゼントにできる？」→ ギフトラッピングの案内
└── 「返品したい」→ 返品ポリシーの説明 + 交換の提案
```
