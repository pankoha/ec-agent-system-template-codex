# Shopify運営最適化 詳細ガイド

## CVR基準値と改善フレームワーク

### 3つの必須KPI
```
Shopify CVRの3層構造:
├── Layer 1: カート追加率
│   ├── 基準値: 最低1%
│   ├── 計算: カート追加数 ÷ セッション数
│   ├── 1%未満の場合:
│   │   ├── LP/商品ページの訴求力不足
│   │   ├── ターゲティングのミスマッチ
│   │   ├── 価格が高すぎる
│   │   └── 商品画像・説明が不十分
│   └── 改善施策: LP改善、ターゲット見直し、価格調整
│
├── Layer 2: 受注率（CVR）
│   ├── 基準値: 最低1%、1.5%で健全、2-3%で優秀
│   ├── 計算: 注文数 ÷ セッション数
│   ├── カート追加率は高いのにCVRが低い場合:
│   │   ├── チェックアウトプロセスの問題
│   │   ├── 決済方法の不足
│   │   └── 送料・納期の不透明さ
│   └── 改善施策: 決済拡充、送料明示、カゴ落ち対策
│
└── Layer 3: カート→購入完了率
    ├── 基準値: 50%
    ├── 計算: 注文数 ÷ カート追加数
    ├── 50%未満の場合:
    │   ├── 送料が想定外に高い（★最大の原因）
    │   ├── 決済方法が少ない
    │   ├── 納期が不明or遅い
    │   ├── 会員登録が必須になっている
    │   └── クーポンコード欄が気になる
    └── 改善施策: 送料無料ライン設定、決済追加、納期明示
```

### CVR改善の優先順位
```
改善ROIが高い順:
├── 1位: 決済方法の拡充（即効性◎、コスト低）
├── 2位: 送料の明示・無料ライン設定（即効性◎）
├── 3位: 納期の明示（即効性◎、コスト零）
├── 4位: カゴ落ちメール設定（即効性○）
├── 5位: 商品画像の充実（中期効果）
├── 6位: レビュー収集（中期効果）
├── 7位: LP構成の改善（中期効果）
└── 8位: デザインリニューアル（長期効果）

★ 核心: デザインを変える前にオペレーション（決済・送料・納期）を整えよ
```

```
決済方法の優先度:
```

## 納期・送料の明示

### 送料戦略
```
送料設定の基本方針:
├── Option A: 全品送料無料（商品価格に含める）
│   ├── メリット: CVR最大化、カート離脱防止
│   ├── デメリット: 低単価商品で利益圧迫
│   └── 推奨: 商品単価¥3,000以上の場合
│
├── Option B: 送料無料ライン設定
│   ├── 例: ¥5,000以上で送料無料
│   ├── メリット: 客単価UP + CVR維持
│   ├── デメリット: ライン未満の注文で離脱リスク
│   └── ★ 推奨: 平均注文額の1.3倍をラインに設定
│
├── Option C: 一律送料
│   ├── 例: 全国一律¥500
│   ├── メリット: シンプルで分かりやすい
│   └── デメリット: 送料が高く見える場合あり
│
└── NG: 地域別送料（離島追加料金等）
    └── チェックアウト時の「想定外の送料」が最大の離脱原因

送料無料ラインの最適化:
├── 現在の平均注文額: ¥____
├── 送料無料ライン: 平均注文額 × 1.3 = ¥____
├── 例: 平均注文額¥3,500 → 送料無料ライン¥4,500
└── 効果: 客単価UP + カート離脱率低下
```

### 納期の明示
```
納期表示のベストプラクティス:
├── 商品ページ: 「ご注文から○営業日以内に発送」
├── カートページ: 「○月○日までにお届け予定」
├── チェックアウト: 配送日時指定オプション
└── 注文確認メール: 発送予定日の明記

★ 納期が不明な商品は買わない
★ 「在庫あり」「即日発送」の表示でCVR向上
```

## 客単価UP施策

### セット販売戦略
```
客単価UPの手法:
├── 1. セット販売
│   ├── 2個セット: 単品×2の10%OFF
│   ├── 3個セット: 単品×3の15%OFF
│   ├── ギフトセット: 本品+ラッピング+メッセージカード
│   └── ★ 「一番人気」ラベルをセット商品に付ける
│
├── 2. 送料無料ライン
│   ├── 「あと¥○○○で送料無料！」の表示
│   ├── カートページに自動表示（アプリで実装）
│   └── 送料無料ラインに合う追加商品を提案
│
├── 3. アップセル（上位商品への誘導）
│   ├── 通常版 → プレミアム版
│   ├── 単品 → 定期便
│   └── 商品ページ内で比較表を表示
│
├── 4. クロスセル（関連商品の提案）
│   ├── 「この商品を買った人はこちらも」
│   ├── カートページで関連商品を表示
│   └── サンクスページでおすすめ商品を表示
│
└── 5. バンドル割引
    ├── 「本品 + アクセサリー」で○%OFF
    └── Shopifyのバンドルアプリを活用
```

### 客単価UP KPI
```
目標設定:
├── 現在の平均注文額（AOV）: ¥____
├── 目標AOV: 現在の1.2〜1.5倍
├── 例: ¥3,000 → ¥3,600〜¥4,500
└── 施策効果の計測: 週次でAOV推移を確認
```

### チェックアウトの離脱対策
```
チェックアウト最適化チェックリスト:
├── □ ゲストチェックアウトを有効化（会員登録不要）
├── □ 住所自動入力を有効化
├── □ 配送日時指定オプションの追加
├── □ ギフトラッピングオプションの追加（該当商品）
├── □ 注文メモ欄の追加（要望記入用）
├── □ 配送料の事前表示（商品ページ上で）
├── □ 「安心のお買い物」バッジ（SSL・返品保証）
├── □ よくある質問リンク
└── □ カスタマーサポート連絡先の表示

Shopify Plus限定（将来検討）:
├── チェックアウトページのカスタマイズ
├── スクリプトによる自動割引
└── カスタムフィールドの追加
```
