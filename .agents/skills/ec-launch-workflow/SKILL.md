# 🚀 新商品ローンチワークフロー（全Agent連携版）

## ワークフロー概要
新商品の市場投入を、PM主導で全Agentが連携する8フェーズで実行。
PMがプロジェクト全体を管理し、各フェーズで専門Agentが担当する。

## 実行手順

### Phase 0: 計画（PROJECT MANAGER主導）
```
├── SMART目標設定
│   ├── KGI: 具体的な数値目標（売上/販売数）
│   ├── KPI: 中間指標（CVR/ROAS/レビュー数）
│   └── 期限: 明確な期日設定
├── スコープ定義（In/Out Scope）
├── WBS作成（全Phase含む）
├── リスク登録簿作成
├── ROI目標設定（300%以上）
└── 関係者へのキックオフ準備
```

### Phase 1: 準備（COMMANDER主導）
```
⚠️ 必須データ収集プロセス:
各Agentからデータを収集してから戦略決定

├── 市場分析
│   ├── 市場規模・成長率
│   ├── ターゲット顧客の定義
│   └── 需要予測（INSIGHT連携）
├── 競合調査
│   ├── 競合商品の分析
│   ├── 価格帯の把握
│   └── 差別化ポイントの明確化
├── ポジショニング
│   └── STP分析
└── 価格戦略
    └── 原価/競合/需要ベースの価格設定
```

### Phase 2: コンテンツ（CREATIVE主導）
```
├── 商品ページ作成
│   ├── 商品説明コピー（PASONA/AIDA）
│   ├── 特徴・ベネフィット
│   └── FAQ
├── 画像制作
│   ├── メイン画像（白背景、商品占有85%+）
│   ├── サブ画像6-7枚
│   └── バナー・広告素材
├── LP作成
│   └── ランディングページ構成
└── SNSコンテンツ
    ├── ティザー投稿
    ├── ローンチ告知
    └── インフルエンサー素材
```

### Phase 3: 集客（ACQUISITION主導）
```
├── SEO対策
│   ├── キーワード設計
│   ├── 商品ページ最適化
│   └── コンテンツSEO
├── 広告設計
│   ├── Google Ads（検索/ショッピング）
│   ├── Meta Ads（認知/リターゲティング）
│   ├── Amazon SP/SB/SD
│   └── 楽天RPP
├── インフルエンサー
│   ├── 候補者選定・スコアリング
│   └── 商品サンプル送付・契約
└── アフィリエイト
    ├── プログラム設定
    └── パートナー告知
```

### Phase 4: 運用（OPERATIONS主導）
```
├── 在庫設定
│   ├── 初期発注量の決定
│   ├── FBA/楽天倉庫/自社の配分
│   └── 安全在庫の設定
├── 価格設定
│   ├── 各チャネル価格
│   ├── ローンチ特別価格
│   └── 価格監視ルール設定
├── 物流手配
│   ├── 配送設定
│   └── 梱包・同梱物準備
└── CS準備
    ├── FAQ準備
    ├── 対応マニュアル
    └── 返品ポリシー設定
```

### Phase 5: 顧客（ENGAGEMENT主導）
```
├── 配信シナリオ
│   ├── ウェルカムシリーズ設定
│   ├── カート離脱リカバリー
│   └── 購入後フォロー
├── 告知メール
│   ├── ティザーメール（1週間前）
│   ├── ローンチ告知（当日）
│   └── リマインド（3日後）
└── LINE配信
    ├── リッチメッセージ準備
    └── 配信スケジュール設定
```

### Phase 6: 計測（INSIGHT主導）
```
├── KPI設定
│   ├── 日次モニタリング指標
│   ├── 週次レビュー指標
│   └── 月次評価指標
├── ダッシュボード
│   ├── リアルタイム売上
│   ├── チャネル別パフォーマンス
│   └── 広告効率
└── 効果測定準備
    ├── A/Bテスト設計
    ├── アトリビューション設定
    └── コホート分析設計
```

### Phase 7: 完了（PROJECT MANAGER主導）
```
├── 進捗管理・リマインド
│   └── 全Phaseの進捗確認
├── 品質チェック（QCD）
│   ├── Quality: スコープ通りか
│   ├── Cost: 予算内か
│   └── Delivery: 期限通りか
├── ROI計算・レビュー
│   ├── 投資額の集計
│   ├── 成果の測定
│   └── ROI 300%達成判定
└── 振り返り・ナレッジ化
    ├── Good/Bad/Next
    ├── 学びの文書化
    └── 次回ローンチへの反映
```

## ローンチチェックリスト
```
□ Phase 0: SMART目標・WBS・リスク登録完了
□ Phase 1: 市場分析・競合調査・価格戦略完了
□ Phase 2: 商品ページ・画像・LP・SNS素材完了
□ Phase 3: SEO・広告・インフルエンサー・アフィリエイト準備完了
□ Phase 4: 在庫・価格・物流・CS準備完了
□ Phase 5: 配信シナリオ・告知メール・LINE準備完了
□ Phase 6: KPI設定・ダッシュボード・効果測定準備完了
□ Phase 7: 品質チェック・ROI計算完了
```
