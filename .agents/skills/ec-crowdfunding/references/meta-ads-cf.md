# Meta広告運用（CF特化）詳細ガイド

## 核心KPI基準値

> **すべてのMeta広告運用はこのKPI基準値で判断する。**

| KPI | 基準値 | 判断 |
|-----|--------|------|
| **CPC** | < ¥30 | 30円超え → クリエイティブ差し替え |
| **CTR** | > 1-2% | 1%未満 → 写真を変更（コピーではなく写真が先） |
| **CVR** | > 2.5% | 2.5%未満 → LP改善（FV・構成の見直し） |
| **CPA** | < ¥1,000 | 1,000円超え → 要改善、1,500円超え → 即座介入 |
| **CPA目標** | **¥600** | 業界ベンチマーク（累計2,000万円超のデータ） |

## 広告予算配分（70/10/10/10）

```
★ 最も重要な法則: 事前集客に70%を投下する

予算配分:
├── 事前集客期: 70%（最重要！ここで勝負が決まる）
├── ローンチ初日: 10%
├── プロジェクト期間中: 10%
└── 終了前ラストスパート: 10%

根拠:
├── 80%成功ルール: 結果の8割は事前集客で決まる
├── 初日売上 × 3倍 = 最終売上（統計的データ）
├── ランキング4位以内 → オーガニック流入の爆発
└── 事前にLINE友達を集めないと初日が作れない
```

## テスト予算と判断基準

```
テスト予算: 約¥40,000

テストの流れ:
├── Step 1: ¥40,000で複数クリエイティブをテスト
├── Step 2: CPA ¥600以下が出るか確認
│   ├── YES → スケール（本予算投下）
│   └── NO → クリエイティブ変更 or 撤退判断
├── Step 3: 勝ちクリエイティブが見つかったら本予算投下
└── Step 4: 50CV到達後に総合判断

介入タイミング:
├── 50CV到達後: 全体のCPA・CTR・CVRを総合判断
├── CPA ¥1,500超え: 即座に介入（写真差し替え最優先）
└── ★ 早すぎる判断は禁物（最低50CVは待つ）
```

## 改善優先順位（最重要）

```
★ 写真 > コピー（ビジュアル変更がCTRに最も効く）

CTRが低い場合（<1%）:
├── 1st: 写真を変更する（最も効果的）
├── 2nd: 動画を試す（15-30秒）
├── 3rd: コピーを変更する
└── ※ コピー改善より写真差し替えの方がインパクトが遥かに大きい

CVRが低い場合（<2.5%）:
├── 1st: ファーストビュー（FV）を変更
├── 2nd: LP本文の構成を見直し
├── 3rd: リターン構成を見直し
└── ※ FVのコピー1行で成果が激変することがある
```

## Instagram中心配信

```
配信面の配分:
├── Instagram: 70-80%（メイン配信面）
│   ├── Stories: 視覚的訴求に最適
│   ├── フィード: 詳細情報の伝達
│   └── リール: 動画クリエイティブ
├── Facebook: 10-20%
└── Audience Network: 0-10%

理由:
├── CFの主戦場であるMakuakeの支援者層とInstagramのユーザー層が重なる
├── ビジュアル訴求力が高い
└── Storiesのスワイプアップ → LINE友達追加の導線が自然
```

## 6段階セットアップ手順

### Step 1: ビジネスマネージャ（BM）作成
```
business.facebook.com にアクセス
├── ビジネスアカウント作成
├── 本人確認の完了
└── 2段階認証の設定（必須）
```

### Step 2: Metaピクセルの設置
```
├── ティザーサイトにピクセルコードを設置（<head>タグ内）
├── 標準イベントの設定
│   ├── PageView: 全ページ
│   ├── ViewContent: 商品詳細閲覧
│   └── Lead: LINE友だち追加
└── ピクセルヘルパーで動作確認
```

### Step 3: 広告アカウントの作成
```
├── 通貨: JPY / タイムゾーン: Asia/Tokyo
├── 支払い方法の設定（クレジットカード）
└── 支出上限の設定
```

### Step 4: Facebookページ + Instagram連携
### Step 5: ドメイン認証（iOS14+対応）
### Step 6: コンバージョン設定（CAPI推奨）

## フェーズ別広告運用

### 事前集客期（予算の70%投下）
```
目的: LINE友だち獲得（CPA ¥600以下）
├── キャンペーン目的: リード獲得 or コンバージョン
├── ターゲティング:
│   ├── 興味関心: 商品カテゴリ関連
│   ├── 年齢/性別: 想定ターゲット層
│   └── 地域: 日本全国（都市部重点）
├── クリエイティブ:
│   ├── 画像: 商品画像 + 「先行割引あり」（★写真が最重要）
│   ├── 動画: 15-30秒の商品紹介
│   └── コピー: 「LINE登録で最大35%OFF」
├── 配信面: Instagram 70-80%
├── 日予算: ¥3,000-10,000
└── 入札: 自動入札（コンバージョン最大化）
```

### ローンチ初日（予算の10%）
```
目的: 支援数最大化 → ランキング上位獲得
├── 予算: 集中投下（通常の3-5倍）
├── 配信先: CFプロジェクトページ直接リンク
└── ターゲティング: LINE友だちリスト + サイト訪問者RT + LAL 1%
```

### 期間中（予算の10%）
```
目的: 継続的な新規獲得
├── 日予算: ¥5,000-20,000
├── クリエイティブ: 2週間ごとに刷新（CTR -30%で判定）
└── ターゲティング: LAL 3-5%まで拡大
```

### 終了前ラストスパート（予算の10%）
```
目的: 駆け込み需要獲得
├── 予算: 中間期の2-3倍
├── クリエイティブ: カウントダウン + 「残りわずか」
└── ターゲティング: 全リターゲティングリスト統合
```

## 4つの売上曲線モデル
```
① 右肩上がり型（理想）: 事前集客成功 + 継続広告 + メディア → 好循環
② 初速型: LINE集客は成功したが新規獲得が弱い → 中間期の広告追加
③ 中だるみ型: 広告疲弊 → 2週間ごとのクリエイティブ刷新
④ 失速型: 市場ミスマッチ → 損切り検討（目標30%未達で戦略転換）
```

## 広告クリエイティブ制作ガイドライン
```
画像広告:
├── サイズ: 1080x1080（フィード）/ 1080x1920（ストーリーズ）
├── テキスト占有率: 20%以下
├── ★ 商品が主役（写真のクオリティが最重要）
└── 割引率・特典を明記

動画広告:
├── 長さ: 15-30秒
├── 最初の3秒で注意を引く
└── 字幕必須（音声OFFでも伝わる）
```
