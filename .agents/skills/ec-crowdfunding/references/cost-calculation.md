# 原価計算 詳細ガイド

## 原価計算表の構造

### 基本原価計算
```
原価構成:
├── 1. 製造原価（COGS）
│   ├── 商品本体: 製造単価 × 数量
│   ├── パッケージ: 化粧箱・緩衝材・取説
│   ├── 付属品: アクセサリー・ケーブル等
│   └── 検品費: 検品単価 × 数量
│
├── 2. 輸送費
│   ├── 国際送料: FOB/CIF/DDPの選択
│   ├── 通関費用: 通関手数料
│   ├── 国内倉庫入庫: ハンドリング費
│   └── 保険料: CIF価格の0.5-1%
│
├── 3. 関税・諸税
│   ├── 関税: CIF価格 × 関税率（HSコードで決定）
│   ├── 消費税: (CIF + 関税) × 10%
│   └── 地方消費税: 消費税 × 22/78
│
├── 4. 国内配送料
│   ├── 支援者への配送: 宅配便/ゆうパック
│   ├── 梱包資材: ダンボール・緩衝材
│   └── ピッキング・梱包作業費
│
├── 5. プラットフォーム手数料
│   └── 支援金額 × 20%（Makuakeの場合）
│
├── 6. 広告費（1支援あたり配賦）
│   └── 広告総額 ÷ 支援件数
│
└── 7. その他
    ├── クリエイティブ制作費（配賦）
    ├── サンプル費用（配賦）
    └── 予備費（原価の5-10%）
```

## 海外送金方法の比較

```
★ WISE（旧TransferWise）が最安

| 送金方法 | 手数料目安 | 為替レート | 推奨度 |
|----------|-----------|-----------|--------|
| WISE | 0.5-1%程度 | 実勢レート（中値） | ★★★★★ |
| 楽天銀行 | ¥1,750/回 | やや不利 | ★★★☆☆ |
| メガバンク（三菱UFJ等） | ¥3,000-7,500/回 | 不利（TTSレート） | ★★☆☆☆ |
| PayPal | 2.5-4% | やや不利 | ★☆☆☆☆ |

WISEのメリット:
├── 為替レートが実勢レート（中値）で最も有利
├── 手数料が最安（0.5-1%程度）
├── 送金速度が速い（通常1-2営業日）
├── 個人事業主でも利用可能
└── ★ 100万円の送金で数千円〜数万円の差が出る
```

## 関税の調べ方

### HSコードの特定
```
Step 1: 商品分類の確認
├── 税関HP「実行関税率表」で検索
│   └── https://www.customs.go.jp/tariff/
├── HSコード体系: 類(2桁) → 項(4桁) → 号(6桁) → 統計細分(9桁)
└── 不明な場合は税関に事前教示を申請

Step 2: 税率の確認
├── 基本税率: WTO加盟国向け
├── 協定税率（MFN）: WTO加盟国に適用
├── 特恵税率（GSP）: 発展途上国に適用
├── EPA/FTA税率: 協定締結国に適用
│   ├── 中国: RCEP（段階的削減）
│   ├── ベトナム: EPA（多くが無税）
│   └── タイ: EPA
└── 目安: 0-15%（商品により大きく異なる）
```

### 関税計算の実例
```
例: 中国製電気製品を100個輸入（税率3.0%）

CIF価格: FOB($1,000) + 運賃($200) + 保険($12) = $1,212
関税: $1,212 × 3.0% = ¥5,454（@¥150）
消費税: (¥181,800 + ¥5,454) × 10% = ¥18,725
地方消費税: ¥18,725 × 22/78 = ¥5,278
合計: ¥29,457 → 1個あたり ¥295
```

## フォワーダー活用

```
★ 個人事業主でもフォワーダーを使える

フォワーダーとは:
├── 国際物流のプロフェッショナル（貿易仲介業者）
├── 輸出入の手続き・通関・輸送を一括代行
├── 個人事業主でも利用可能（法人である必要なし）
└── 大量輸送で個口配送より大幅にコスト削減

利用メリット:
├── 通関手続きの代行（専門知識不要）
├── 物流コストの大幅削減（スケールメリット）
├── 保険・追跡・品質管理の一元化
└── トラブル対応（税関でのストップ等）も代行

配送方法の選択:
├── 航空便: 早い（3-7日）が高い → 少量・急ぎの場合
├── 船便: 安い（14-30日）が遅い → 大量輸送の場合
└── ★ CFの場合は船便が多い（コスト優先）
```

## EC専用倉庫

```
★ 自宅保管ではなくEC専用倉庫を使う

オープンロジ:
├── 初期費用: なし（従量課金制）
├── 入庫費: 商品サイズにより ¥15-50/個
├── 保管費: ¥3-10/個/日
├── 出荷費: ¥300-600/件
├── 特徴: API連携あり、Shopifyとの自動連携
└── ★ 小規模から始められる（初期投資なし）

国立倉庫:
├── 大量在庫向けの物流センター
├── パレット単位での保管
├── 検品・梱包・出荷まで一括対応
└── ある程度の物量がある場合に有利

倉庫選定基準:
├── 初回ロット: 100-500個 → オープンロジ推奨
├── 大量ロット: 500個以上 → 国立倉庫検討
└── ★ 自宅保管は品質管理・スペースの問題でNG
```

## 為替リスク管理
```
├── 基準レート: 発注時の実勢レートを使用
├── バッファ: +5-10%の為替バッファを原価に含める
├── 例: 実勢¥150/$ → 原価計算は¥160/$で計算
├── 為替予約: 大量発注時は為替予約を検討
└── チェック: 仕入先通貨(USD/RMB)・決済タイミング・変動リスク
```

## MOQ（最小発注量）と価格交渉
```
初回発注量の決定:
├── CFの目標支援数 + バッファ20-30% + 不良率5%
├── 例: 目標500個 → 初回発注650個

サプライヤー交渉:
├── 3社以上から見積もり取得
├── 「初回はテスト発注として○個で」と提案
├── 数量別の価格テーブルを要求
├── 支払い条件の柔軟性で単価交渉
└── サンプル3-5個は無料交渉可
```

## 発注管理ドキュメント
```
POフォーマット（Purchase Order）:
├── PO番号 / 発注日 / 発注者情報 / サプライヤー情報
├── 商品明細（品名/型番/仕様/数量/単価/合計）
├── 納期 / 配送条件(FOB/CIF/DDP) / 支払い条件 / 品質基準
└── 署名欄

ケースマーク（外箱表示）:
├── 荷受人名 / 仕向地 / 原産国 / ケース番号
├── 総個数 / 重量(G/N) / サイズ / 取扱注意マーク
└── ★ 通関時に必ず確認される

輸出書類:
├── インボイス / パッキングリスト / B/L or AWB
├── 原産地証明書（EPA/FTA適用時）
└── 検査証明書（PSE認証 / 技適マーク / 食品衛生法 等）
```

## 収支シミュレーション
```
3パターンシミュレーション:

悲観シナリオ（支援100件）: 赤字 → 広告予算の見直し
標準シナリオ（支援300件）: 利益率15-20% → 合格ライン
楽観シナリオ（支援600件）: 利益率25%+ → スケールメリット

損益分岐点:
BEP = 固定費 ÷ (平均売価 - 変動費/個)
例: ¥500,000 ÷ (¥8,000 - ¥4,500) = 143個で黒字化
```
