# インフルエンサー & PR 詳細ガイド

## 核心: 第三者の権威で信頼構築

> **自分で「いい商品です」と言うより、第三者が言う方が100倍信頼される。**
> インフルエンサー・PR TIMES・プロセスエコノミーの3本柱で信頼を構築する。

## YouTuber活用戦略

### 成功事例: Zip Tarp（広告費0円 → 300万円売上）
```
Zip Tarpの事例:
├── 広告費: ¥0（インフルエンサーのみ）
├── 売上: 300万円+
├── 方法: ニッチ系YouTuberとのコラボ
├── ★ 相性の良いインフルエンサーなら広告なしで成功できる

成功のポイント:
├── ニッチ系（フォロワー数万人規模）の方が相性が良い
├── メガインフルエンサー（100万人超）は費用対効果が悪い
├── 「レビュー依頼」ではなく「共創・コラボ」の姿勢
└── 商品を無償提供 → 正直なレビュー依頼（ステマではない）
```

### インフルエンサー選定基準
```
理想的なインフルエンサー:
├── フォロワー: 1万〜10万人（ニッチ系が最適）
├── ジャンル: 商品カテゴリと完全一致
├── エンゲージメント率: 3%以上（いいね÷フォロワー数）
├── 視聴者層: 自社ターゲットと一致
└── 世界観: ブランドイメージと親和性がある

避けるべきインフルエンサー:
├── フォロワー数だけが多い（買いフォロワーの疑い）
├── 案件投稿ばかり（信頼性が低い）
├── ジャンルが合わない（無関係なインフルエンサー）
└── ネガティブな過去がある（ブランドリスク）
```

### インフルエンサー連絡テンプレート
```
件名: 「[ブランド名] × [チャンネル名]のコラボレーションのご相談」

本文:
こんにちは。[ブランド名]の[名前]と申します。

いつも[チャンネル名]を拝見しており、[具体的な動画名]の
[具体的なポイント]が特に印象的でした。

現在、[商品カテゴリ]の新商品をMakuakeでローンチ予定です。
[商品の1行説明 + PODを1つ]

つきましては、商品をお送りして
実際にお使いいただいたレビューをお願いできないでしょうか。
（費用・掲載方法等は柔軟にご相談できます）

ご興味をお持ちいただけましたら、
詳細資料をお送りいたします。

何卒よろしくお願いいたします。

★ ポイント:
├── 相手のコンテンツを具体的に言及する（テンプレ感を消す）
├── 商品のPODを1行で簡潔に伝える
├── 条件は柔軟に相談できると伝える
└── 詳細資料は後送（最初のメールを短くする）
```

### 報酬体系の目安
```
フォロワー1万人: 商品無償提供のみ or 1-3万円
フォロワー3万人: 3-10万円
フォロワー5万人: 5-15万円
フォロワー10万人: 10-30万円

★ CF案件の場合:
├── 「成功報酬型」（売上の○%）を提案するのもあり
├── アフィリエイトリンクでのトラッキング
└── 広告費0円で始められるため、リスクが低い
```

## プロセスエコノミー

### 概念
```
プロセスエコノミーとは:
├── 完成品ではなく「制作過程」を発信してファンを増やす手法
├── 応援者を「共創者」として巻き込む
├── CF開始前からエンゲージメントを構築
└── ★ 「この商品を応援したい」という気持ちを醸成する

発信内容:
├── 開発ストーリー: なぜこの商品を作るのか
├── 試行錯誤: 失敗談や改善過程も包み隠さず
├── 製造過程: 工場・サプライヤー訪問の様子
├── テスト結果: 品質テストの過程と結果
├── チームの想い: メンバー紹介や日常
└── ユーザーの声: モニターのフィードバック

発信チャネル:
├── Instagram Stories/リール（日次）
├── Twitter/X（日次）
├── note（週次）
├── YouTube（隔週）
└── ★ 全チャネルでLINE友だち追加への導線を設置
```

### 効果
```
├── CF公開前からファンコミュニティが形成される
├── 支援は「購入」ではなく「応援」になる
├── SNSシェア率が大幅に向上（自分の推しプロジェクト感）
├── LINE配信の開封率・転換率が向上
└── ★ 広告だけに頼らない「有機的な集客」が実現する
```

## PR TIMES活用

### 基本情報
```
PR TIMES:
├── 日本最大級のプレスリリース配信サービス
├── 月額: 無料プラン or 有料プラン（配信回数による）
├── 配信先: 300媒体以上のメディアに一括配信
├── SEO効果: PR TIMESのドメインパワーが非常に高い
└── ★ Googleで「商品名」を検索した時にPR TIMESの記事が上位表示される

配信タイミング:
├── CFローンチの1-2日前（事前告知として）
├── ローンチ初日（実績報告として）
├── 目標達成時（ニュース性がある）
└── ストレッチゴール達成時
```

### プレスリリースの書き方
```
構成:
├── タイトル: 「新規性 + 数値」（目を引くキーワード）
│   └── 例:「世界初の○○素材を採用した△△、Makuakeで先行販売開始」
├── リード文: 5W1Hで概要を簡潔に（3-4行）
├── 商品紹介: PODを中心に特徴を3つ
├── 開発ストーリー: なぜ作ったか（共感を生む）
├── エビデンス: 第三者試験結果・特許情報
├── リターン案内: 割引率・限定数
├── プロジェクト情報: URL・期間・目標金額
└── 会社/チーム情報

★ ポイント:
├── タイトルに具体的な数値を入れる
├── 画像を3-5枚添付（記者が記事に使いやすい）
├── 「世界初」「日本初」「業界初」があると取り上げられやすい
└── 配信は火〜木の10時がメディアに読まれやすい
```

### SEO効果
```
PR TIMESのSEO効果:
├── ドメインパワーが非常に高い（Googleでの信頼性）
├── 「ブランド名」「商品名」で検索すると上位表示される
├── メディア記者がPR TIMESを情報源にしている
├── 取材依頼のきっかけになることが多い
└── ★ 無料で大きなSEO効果が得られる最高のツール
```

## Instagram Storiesカウントダウン

### 設定方法
```
CF公開日のリマインド:
├── Instagram Storiesの「カウントダウンスティッカー」を使用
├── 「○月○日 ○時 プロジェクト開始！」と設定
├── フォロワーが「リマインドを受け取る」をタップ
├── 公開日にInstagramからプッシュ通知が届く
└── ★ LINE以外のリマインドチャネルとして効果的

活用方法:
├── CF公開の1-2週間前からカウントダウン開始
├── 毎日のStoriesでカウントダウンスティッカーを付ける
├── ストーリーの内容: 日替わりで商品の特徴を紹介
├── スワイプアップ or リンクスティッカーでLINE登録へ誘導
└── ★ Instagram → LINE → CF支援 の導線を構築
```

## 施策の優先順位
```
費用対効果の順:
├── 1st: プロセスエコノミー（コスト: ¥0、効果: 高）
│   └── 毎日の発信の手間だけ。最もROIが高い
├── 2nd: PR TIMES配信（コスト: ¥0-3万円、効果: 中-高）
│   └── SEO効果 + メディア露出のきっかけ
├── 3rd: Instagram Stories（コスト: ¥0、効果: 中）
│   └── フォロワーへのリマインド効果
├── 4th: ニッチ系YouTuber（コスト: ¥0-10万円、効果: 高）
│   └── 商品無償提供のみでOKなケースもある
└── ★ すべて「Meta広告の補完」として位置づける
    └── メイン集客はMeta広告→LINE, 上記は追加ブースト
```
