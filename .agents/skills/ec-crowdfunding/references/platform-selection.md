# プラットフォーム選定 詳細ガイド

## 基本方針: Makuake一択

> **売上 = 流入ユーザー数 × CVR（成約率） × 顧客単価**
> 失敗の最大原因は「流入ユーザー数」の不足。プラットフォームの自社集客力が売上を決定する。

## Makuakeが最強な5つの理由

### 1. 審査が最も厳しい → 品質の信頼性
```
厳格な審査 → 出品商品の品質担保 → ユーザーの信頼が高い → CVRが高い
```

### 2. SNSアカウントのリスト数
```
圧倒的なフォロワー基盤でオーガニックリーチが大きい
```

### 3. 近似プロジェクトからの流入
```
アルゴリズムによる回遊で「似た商品を見た人」が自動的に流れてくる
```

### 4. メルマガ配信（会員236万人）
```
メルマガ配信の種類:
├── 新着プロジェクト配信
├── カテゴリ特化配信
├── 週間ランキング配信
└── ラストチャンス配信（終了間際）

★ 会員236万人への直接配信。これだけで2-3万PVが得られる
```

### 5. リターゲティング広告（プラットフォーム負担・無料）
```
Makuakeがリタゲ広告費用を負担:
├── ページを見たが購入しなかったユーザーに自動追跡
├── 売上の10-30%がリタゲ経由
├── 出品者の追加コスト: ¥0
└── ★ CAMPFIREにはこの機能がないため非推奨
```

## 公開曜日・時刻の選び方
```
★ 火曜日 11時がベスト

理由:
├── 売れている状態で土日を迎える → 週末にオーガニック流入が増加
├── メルマガ枠が新着・カテゴリで確保しやすい
├── 11時公開 → スタートダッシュクーポンの有利性
└── 午前中に公開 → 夕方のメルマガに掲載される可能性

避けるべき:
├── 土日 → メルマガ枠が週間ランキング/ラストチャンスのみ
├── ※ 事前集客で十分なリストがある場合は土曜日でもOK
└── 金曜 → 週末前で購買意欲が分散

11時公開の重要性:
├── Makuakeはスタートダッシュクーポンの仕組みがある
├── 11時に合わせてLINE配信 → 一斉購入 → ランキング上位
└── 「もうすぐ開始」機能で事前通知登録者を獲得可能
```

## 「もうすぐ開始」機能の活用
```
Makuake審査通過後に利用可能:
├── プロジェクト公開前に「もうすぐ開始」ページが生成される
├── ユーザーが「通知を受け取る」ボタンで事前登録できる
├── 公開日にMakuakeから自動通知が送られる
├── 事前登録数が多いほどランキング初動に有利
└── ★ Meta広告のLPとして「もうすぐ開始」ページも使える
```
