# Free Source Map

Use this reference when selecting a free or low-cost source.

| Need | Preferred source | Free constraint | Notes |
|---|---|---|---|
| YouTube channel/video stats | YouTube Data API | Default quota; search is expensive | Prefer known channel/video IDs. |
| Public page verification | Codex Chrome plugin/direct Chrome access | User must have it available | Good for screenshots and manual-style checks. |
| YouTube transcripts | Public transcript tools or page extraction | Availability varies | Respect rights and quote limits. |
| Threads public posts | Bright Data MCP or public web | Free request cap | Public data only. |
| X/Twitter public posts | Bright Data MCP or public web | Free request cap; instability likely | Verify important claims manually. |
| Instagram public profile/post sampling | Apify/Bright Data/public web | Small sample only | Avoid login-gated/private data. |
| TikTok public trend sampling | Apify/public web | Small sample only | Metrics can change quickly. |
| Owned account analytics | User export CSV/XLSX | User must provide export or auth | Prefer exports for accuracy. |

If a source is unavailable, state the fallback and reduce confidence.
